## 2. 질문 목록 (14문항)

> 원칙: LP 제작에 직접 필요한 질문만. 웹 조사로 확인된 사항은 "맞는지 확인"으로 대체.

### A. LP 콘텐츠 확인 (4문항)

웹 조사로 브랜드 스토리, USP, 매장 현황 등을 파악했습니다. 아래는 LP에 직접 반영할 내용의 확인/보완입니다.

| # | 질문 | LP 반영 위치 | 비고 |
|---|------|-------------|------|
| A1 | 저희가 조사한 브랜드 스토리(하노이 50년 전통, 13시간 육수, 80% 베트남인 고객)가 LP에 쓰기에 맞나요? 수정/추가할 부분이 있으면? | 히어로 + 브랜드 스토리 섹션 | 웹 조사 내용 확인용 |
| A2 | 현재 정확한 매장 수는? (웹사이트 8개 + 제천 + 미아 = 약 10개로 파악) | 히어로 "전국 N개 매장" | 숫자 확인 |
| A3 | 가맹 모집 대상은 쌀국수 단일 브랜드인가요, 국밥/반미/카페도 포함인가요? | LP 범위 결정 | 멀티 브랜드 운영 중이므로 |
| A4 | 모집 대상 지역은? 전국 확장인가요, 특정 지역 우선인가요? | LP 메시지 방향 | 현재 부산/경남 집중 |

### B. LP에 넣을 데이터 범위 (4문항)

| # | 질문 | LP 반영 위치 | 비고 |
|---|------|-------------|------|
| B1 | 투자비 구성(가맹비, 교육비, 인테리어 등)과 **LP에 공개 가능한 범위**는? | 투자 비용 섹션 | 경쟁사 대부분 비공개 → 공개 시 차별화 |
| B2 | 매출/수익 관련 데이터 중 **LP에 표시해도 되는 것**은? (예: 평균 매출, 수익률 등) | 수익 모델 섹션 | "공개 가능 범위"가 핵심 |
| B3 | 소형 매장이나 배달 특화 모델이 있나요? 있다면 최소 평수는? | 소자본 창업 섹션 유무 | 없으면 해당 섹션 제외 |
| B4 | 교육 기간과 SV 지원(방문 주기 등)의 구체적 내용은? | 교육/지원 섹션 | 개요는 확인됨, 상세만 |

### C. LP에 쓸 자산 (2문항)

| # | 질문 | LP 반영 위치 | 비고 |
|---|------|-------------|------|
| C1 | 브랜드 가이드라인(로고, 컬러, 폰트), 고해상도 매장/음식 사진, 점주 인터뷰 영상 등 LP에 쓸 소재를 정리해서 공유 가능한가요? | 전체 디자인 + Social Proof | 이홍렬TV/부산일보 외 추가 자료 |
| C2 | LP에서 가장 강조하고 싶은 메시지 1~3가지는? | 히어로 + 전체 톤 | 조사 결과 "13시간 육수 + 베트남인 인정 + 원칙" 제안 예정 |

### D. LP 기능/기술 (4문항)

| # | 질문 | LP 반영 위치 |
|---|------|-------------|
| D1 | 문의 폼에 어떤 정보를 받고 싶나요? (이름, 연락처, 희망 지역 등) | 상담 신청 폼 |
| D2 | 문의 접수 후 프로세스는? (자동 응답 → 담당자 연결? 전화 상담?) | CTA 후 퍼널 |
| D3 | 기존 웹사이트(raeoakfood.co.kr)나 다른 도구와 연동이 필요한가요? | 기술 요구사항 |
| D4 | 런칭 희망 일정과 예산 범위는? | 프로젝트 스코프 |

**총 14문항** — 미팅 시간 절약을 위해 웹 조사 결과를 먼저 공유한 후, 확인/보완 위주로 진행

## 3. 사전 리서치 공유 (미팅 초반 15분)

미팅 시작 후 "저희가 사전에 조사한 내용"으로 시작하여 전문성을 보여주면서 자연스럽게 확인 질문으로 연결합니다.

### 공유 1: 라이옥 브랜드 이해

> "하노이 50년 전통 라이옥에서 기술을 전수받아 2015년 대연동에서 시작, 13시간 이상 끓이는 육수, 고객의 80%가 베트남인이라는 점이 인상적이었습니다. LP에 이 내용을 핵심 스토리로 쓰려고 하는데, 수정하거나 추가할 부분이 있으신가요?"

→ A1으로 자연스럽게 연결

### 공유 2: 경쟁 환경 + 투명성 전략 제안

> "주요 경쟁사 10곳을 분석했는데, 포메인만 가맹 전용 LP가 있고 나머지는 없었습니다. 또 대부분 투자비를 공개하지 않는데, 라이옥이 투자비 범위를 공개하면 '정보를 숨기지 않는 프랜차이즈'로 차별화할 수 있습니다."

→ B1, B2로 연결 ("공개 가능한 범위가 어디까지인지?")

### 공유 3: 라이옥만의 강점

> "13시간 육수, 매일 5종류 이상, 원칙에 어긋나면 판매 중단 — 이런 스토리가 LP에서 강력한 설득 포인트입니다. 특히 '베트남인 80%가 고객'이라는 건 다른 브랜드에 없는 정통성 근거입니다."

→ C2로 연결 ("이 중 LP에서 가장 강조하고 싶은 것은?")

## 5. 미팅 중 유의사항

### Do (해야 할 것)

| 항목 | 이유 |
|------|------|
| 리서치 결과를 **질문 형태**로 제시 | "저희가 조사한 바로는 ~인데, 실제로는 어떤가요?" — 전문성 + 경청 |
| **숫자**로 근거 제시 | "치킨 3만개 대비 쌀국수 600개" — 구체적 데이터가 설득력 |
| 경쟁사 랜딩페이지 **캡처**를 보여주며 설명 | 시각적 레퍼런스가 방향 합의에 효과적 |
| **미결 사항**을 명시적으로 기록 | "이건 자료 확인 후 결정"으로 넘기고, 미결 리스트 유지 |
| 클라이언트의 **직감/경험** 존중 | 데이터와 현장 경험이 충돌하면 현장이 우선일 수 있음 |

### Don't (하지 말 것)

| 항목 | 이유 |
|------|------|
| 경쟁사를 **비하**하지 않기 | "미스사이공이 망했다"가 아니라 "급속 확장의 리스크 사례" |
| 구체적 **매출 약속**을 하지 않기 | 정보공개서 외 데이터로 수익을 보장하면 법적 리스크 |
| **기술 용어** 남용하지 않기 | CTA, 퍼널, A/B 테스트 → "문의 버튼", "고객 동선", "비교 실험" |
| 일정을 **너무 빡빡하게** 잡지 않기 | 여유를 두어 품질 확보 |

## 7. 클라이언트 사전 공유용 요약 (1페이지)

> 미팅 2일 전, 클라이언트에게 아래 내용을 요약하여 공유합니다.  
> 목적: "이 팀이 준비를 잘 해왔구나"라는 인상 + 미팅 방향에 대한 사전 합의

### 쌀국수 프랜차이즈 가맹점 모집 랜딩페이지 — 사전 리서치 요약

**1. 시장 기회**
- 에스닉 푸드 매장 9년간 5배 성장, 외식업 경기전망지수 1위 카테고리
- 쌀국수 프랜차이즈 전체 600~700개 수준 — 치킨(3만+), 커피(4만+) 대비 초기 성장 단계
- 건강식(글루텐프리, 저칼로리), 1인 가구(815만), 배달 시장(40조) 트렌드와 높은 정합성

**2. 경쟁 환경**
- 주요 브랜드: 미분당(171개), 포메인(100개), 미스사이공(90개) 등 10여 개
- 가맹 전용 독립 랜딩페이지 운영 브랜드는 포메인 1곳뿐
- 대부분 가맹비/창업비용을 웹상에 비공개 → **투명성이 곧 차별화**

**3. 타겟 가맹점주**
- 핵심 타겟: 30~50대 탈직장인 (전체 창업의 71.4%, 투자금 1~2억 보유)
- 의사결정 기준: ① 수익 안정성 ② 본사 신뢰도 ③ 운영 난이도
- 핵심 전환 장벽: 배우자/가족 설득, 투자금 손실 두려움

**4. 랜딩페이지 방향 (가설)**
- 모바일 퍼스트, 가맹 전용 독립 페이지
- 투자비 범위 + 정보공개서 기반 매출 데이터 투명 공개
- 실제 가맹점주 인터뷰 + 수익 사례 중심 설득
- 간편 상담 신청 폼 → 자동 안내 → 1:1 상담 연결 퍼널

**미팅에서 논의하고 싶은 것**
- 브랜드 차별화 포인트와 핵심 메시지 방향
- 투자비/매출 데이터 공개 가능 범위
- 현재 가맹 모집 현황과 문의 데이터
- 런칭 일정과 우선순위
