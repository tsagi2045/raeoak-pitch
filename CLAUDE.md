# RICE-NOODLE-FRANCHISE

쌀국수 프랜차이즈 가맹점 모집 — 시장 조사, 경쟁사 분석, 랜딩페이지 제작

## 디렉토리 구조

- `raw/` — 불변 원본 소스 (LLM은 읽기만, 수정 금지)
- `wiki/` — LLM이 컴파일한 지식 위키
  - `sources/` — 원본별 구조화 요약
  - `concepts/` — 도메인 개념 페이지
  - `entities/` — 사람, 조직, 브랜드
  - `output/` — Query 결과물
- `docs/` — 기존 참고 문서 (읽기 전용)

## Rules

위키 운영 규칙은 `rules/wiki-schema.md`를 참조할 것.
