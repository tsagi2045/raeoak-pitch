# 소자본 창업

MZ세대를 중심으로 한 소자본 프랜차이즈 창업 트렌드와 쌀국수 프랜차이즈에의 시사점.

## 창업 시장 현황

| 지표 | 수치 | 출처 |
|------|------|------|
| 프랜차이즈 평균 창업비용 | 1억 3,242만원 | 중소기업중앙회 2025 |
| 인테리어/설비 비중 | 85% (1.13억원) | 공정위 2024 통계 |
| 투자금 회수 기간 | 평균 31.4개월 | KBC 2025 |
| 예비 창업자 5천만원 이하 | 29.6% | 마이프차 2026 리포트 |
| 예비 창업자 1억원 이하 | 64.3% | 마이프차 2026 리포트 |

## 2026년 창업 키워드

마이프차 2026 트렌드 리포트 기준:
- **가성비** (112회 언급) — 1위
- **경쟁** (96회) — 2위
- **무인** (84회) — 3위

가장 관심 있는 아이템: 커피(28.6%), 무인(19.3%), 비식음 서비스(12.6%)

## 쌀국수 프랜차이즈의 소자본 적합성

| 장점 | 상세 |
|------|------|
| 소형 매장 가능 | 15~20평으로 운영 가능. 인테리어비 절감 |
| 1~2인 운영 | 조리 단순화(육수/소스 본사 공급, 면 삶기 중심) |
| 낮은 원가율 | 면류 특성상 식재료 원가 통제 용이 |
| 배달 특화 모델 | [[포메인]] RED 모델처럼 초소형/공유주방 옵션 |

## 쌀국수 브랜드별 창업비용

| 브랜드 | 총 창업비용 | 비고 |
|--------|-----------|------|
| 헬로베트남 | 770만원 | 초저비용 |
| **[[라이옥]] 라이트** | **~3,200~4,200만원** | **10~15평, 1인 운영 가능** |
| 포솔 (라이옥 자매) | 5,300만원 | 10평, 1인 특화 |
| [[미스사이공]] | ~9,600만원 | 업계 평균 |
| **[[라이옥]] 스탠다드** | **~6,800~8,800만원** | **25평+, 풀 운영** |
| [[미분당]] | 1억 92만원 | 중간 |
| 월미당 | 1억 300만원 | 소자본 프리미엄 |
| [[에머이]] | 1억 2,403만원 | 중상 |
| 반포식스 | 1억 8,540만원 | 프리미엄 |

→ 라이옥 라이트는 업계 최저가 수준 (헬로베트남 제외). 예비 창업자 29.6%가 원하는 "5천만원 이하" 구간에 적합.

## 타겟 페르소나와의 연결

- **페르소나 C (이준혁, 28세)**: 가용 자산 7천만원, 본사 밀착 지원 필요
- 의사결정 기준 1순위: 초기 투자 비용(35%)
- 핵심 설득 메시지: "처음이라서 괜찮습니다"
- CTA: "무료 창업 상담 신청"

## 라이옥 라이트 모델 상세

브로슈어로 확인된 라이옥의 소자본 모델:

| 항목 | 라이트 (10~15평) |
|------|---------------|
| 가맹금 | 500만 |
| 교육비 | 300만 |
| 인테리어 | 200만/평 |
| 주방설비 | 1,000~1,200만 |
| 월 예상 영업이익 | 680만 (매출 2,000만 기준, 34%) |
| 로열티 | 30만/월 |

→ 페르소나 C(이준혁, 28세, 가용 자산 7천만원)에 최적. "내 자본으로 가능한가?" 질문에 명확히 답변 가능.
