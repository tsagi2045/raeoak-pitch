# 랜딩페이지 전략

쌀국수 프랜차이즈 가맹점 모집 랜딩페이지의 벤치마킹 결과와 전략 방향.

## 경쟁사 랜딩페이지 현황

| 브랜드 | 형태 | 특징 |
|--------|------|------|
| **[[포메인]]** | 가맹 전용 독립 도메인 (phomeinfc.com) | **유일한 독립 LP**. AI 입지분석, RED 모델 소개 |
| [[미분당]] | 홈페이지 내 서브 섹션 | 마이프차 플랫폼 활용 |
| 호아빈 | 홈페이지 내 창업안내 | 수익 데이터 전면 노출 (월 순수익 1,000만원) |
| 월미당 | 홈페이지 + 블로그 | 공격적 수치 마케팅 (순수익률 40%, 회수 3개월) |
| [[미스사이공]] | 기본 웹 | 창업자 스토리텔링 |
| 나머지 | 홈페이지 내 서브 | 비용 비공개, 상담 유도 |

## 차별화 전략 5가지

1. **가맹 전용 독립 랜딩페이지** — 포메인 외 부재. 독립 도메인 or 서브도메인
2. **비용 투명성** — 투자비 구간 공개(예: "8,000만~1.2억"). 대부분 비공개인 시장에서 신뢰 구축
3. **데이터 기반 수익 사례** — 정보공개서 기반 평균 매출 + 실제 점주 영상 인터뷰
4. **디지털 가맹 퍼널 자동화** — 랜딩 → 관심 등록 → 자동 안내 → 상담 예약
5. **AI 상권 분석 미리보기** — 주소 입력 → 유동인구/경쟁점/예상 매출 즉시 제공

## 전환 퍼널 설계

고객 여정에서 랜딩페이지의 핵심 구간은 **탐색→관심 전환** (2→3단계).

```
광고 노출 → LP 도달 → 관심 등록(리드) → 상담 완료 → 설명회 → 계약
```

| 단계 | 전환율 벤치마크 |
|------|---------------|
| LP 전환율 (방문→리드) | 5~10% (B2B 중앙값 6.6%) |
| 리드→상담 완료 | 30~40% |
| 상담→설명회 | 50~60% |
| 설명회→계약 | 50~60% |
| 전체 리드→계약 | 약 2.3% (글로벌 평균) |

초기 현실적 목표: **월 3~8건 계약**

## 페르소나별 CTA

| 타겟 | CTA | 톤 |
|------|-----|-----|
| 탈직장인 (Primary) | "수익 시뮬레이션 받기" | 전문적, 데이터 중심 |
| 다점포 운영자 | "가맹점 실적 자료 요청" | 비즈니스적, 동업자 |
| 소자본 첫 창업 | "무료 창업 상담 신청" | 친근, 응원 |

## 섹션 구성 (제안)

히어로 → 시장 기회 → 수익 모델 → 운영 시스템 → 교육/지원 → 가맹점 성과 → 투자 비용 → 다점포 혜택 → CTA

## 핵심 전환 장벽과 해소 전략

| 장벽 | 해소 전략 |
|------|----------|
| 투자비 불확실성 | 비용 범위 투명 공개 |
| 수익성 의심 | 정보공개서 기반 데이터 + 점주 인터뷰 |
| 본사 신뢰 부족 | 교육/SV/물류 체계 상세 설명 |
| 배우자/가족 설득 | "가족과 함께 보기" 자료, 인쇄용 요약 |
| 정보 과부하 | 단계별 가이드, 1페이지 핵심 요약 |

## 미팅에서 확인 필요

- 독립 도메인 vs 기존 사이트 내 랜딩 선호
- 투자비/매출 데이터 공개 가능 범위 (질문 D5)
- 문의 폼 필드 구성 (질문 E2)
- 문의 후 프로세스 — 자동 응답 → 담당자 연결 (질문 E3)
