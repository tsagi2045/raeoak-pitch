# 배달 시장

한국 온라인 음식배달 시장의 현황과 쌀국수 프랜차이즈에 대한 시사점.

## 시장 규모

| 지표 | 수치 |
|------|------|
| 온라인 음식배달 거래액 (2025) | **40조원** 돌파 (역대 최대) |
| 외식 시장 내 배달 비중 | 2019년 18% → 2022년 30%대 → 2029년 40% 전망 |
| 배달앱 시장 점유율 | 배민 59.2%, 쿠팡이츠 21.0%, 요기요 16.2% |

## 쌀국수의 배달 적합성

| 요인 | 설명 |
|------|------|
| **포장 용이** | 면류 특성상 포장이 간편. 국물/면 분리 포장 가능 |
| **맛 유지** | 식어도 상대적으로 맛이 유지됨 |
| **객단가** | 배달 주문 시 사이드 메뉴 추가 → 객단가 상승 |
| **매출 분산** | 점심 피크 외 저녁/야식 배달로 매출 시간대 분산 |
| **매장 효율** | 매장 매출 + 배달 매출의 이중 수익 구조 가능 |

## 배달 특화 모델 사례

- **[[포메인]] RED 모델**: 딜리버리 특화 소형 매장, 공유주방 입점 옵션
- 초소형 면적으로 인테리어비 절감 → [[소자본-창업]] 연결
- "배달의쌀국수" 같은 배달 전문 브랜드도 등장

## 위협: 배달 수수료

- 배달앱 수수료 = 중개수수료 + 결제수수료 + 광고비
- 영업이익을 압박하는 주요 요인 중 하나
- 자체 배달 인력/채널 확보가 장기적 과제

## 랜딩페이지 시사점

- "매장 + 배달 이중 수익 구조" 메시지로 수익성 강조
- 배달 특화 모델이 있으면 소자본 창업자에게 어필
- 배달 매출 비중, 배달 주문건수 등 데이터를 클라이언트로부터 확보 필요
