# 시장 기회

쌀국수 프랜차이즈 가맹점 모집 관점에서의 기회 요인과 위협 요인을 정리한다.

## 기회 요인

| 기회 | 근거 데이터 |
|------|-----------|
| **블루오션** | 치킨 3만+, 커피 4만+ 가맹점 vs 쌀국수 전체 600~700개. 에스닉 푸드 매장 9년간 5배 성장 |
| **건강식 정합성** | 글루텐프리, 저칼로리(300~400kcal), 저속노화 트렌드에 부합 |
| **1인 가구 확대** | 815만 가구 돌파(2025), 혼밥 긍정률 83%. 쌀국수 = 대표 혼밥 메뉴 |
| **[[배달-시장]] 성장** | 온라인 음식배달 40조원(2025). 쌀국수는 배달 적합성 높음 |
| **운영 효율** | 조리 단순화(육수/소스 본사 공급), 1~2인 운영 가능, 면류 원가율 낮음 |
| **[[소자본-창업]] 수요** | MZ세대 소자본 창업 트렌드 + 소형 매장(15~20평) 운영 가능 |
| **메뉴 확장성** | 분짜, 반미, 볶음면, 월남쌈 등 사이드로 객단가 상승 가능 |
| **경쟁사 랜딩 부재** | [[포메인]]만 독립 랜딩페이지 보유 → 잘 만든 LP 하나로 차별화 |

## 위협 요인

| 위협 | 근거 데이터 |
|------|-----------|
| **인건비/식재료비 상승** | 외식업 영업이익률 12.1%→8.7%(2020~2024). 비용 46.7% 증가 |
| **경쟁 브랜드 증가** | 2024~2025 신규 브랜드 진입 가속. 브랜드 수 50개 이상 |
| **급속 확장 리스크** | [[미스사이공]] 274개→90개 급락 사례. 질적 성장 필요 |
| **외식업 전반 둔화** | 전체 브랜드 수 첫 감소세. 양극화 심화 |
| **가맹사업법 규제 강화** | 2025년 개정: 직영점 3개 이상 필수, 정보 투명성 의무 |
| **계절성** | 여름철 매출 하락 가능성 (냉면류 보완 필요) |
| **배달 수수료 부담** | 배달앱 수수료(중개+결제+광고)가 영업이익 압박 |

## 시장 규모 맥락

- TAM(전체 외식 프랜차이즈 가맹 시장): **3.55조원**/년
- SAM(쌀국수/아시안 면류): **478억원**/년 — TAM의 1.3%
- 쌀국수 점유율 0.35% → **성장 여지가 매우 큼**

## 랜딩페이지 활용 방향

1. **"아직 시작 단계"** 메시지로 시장 성장성 강조
2. 건강식/1인 가구/배달 트렌드 데이터를 설득 근거로 활용
3. 위협 요인은 "본사가 어떻게 대응하는가"를 보여주는 기회로 전환
