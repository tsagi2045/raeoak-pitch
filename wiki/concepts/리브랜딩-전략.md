# 리브랜딩 전략 (라이트 모델)

라이옥 라이트 모델(10~15평, 카운터석, 고회전 소형 매장) 대상의 리브랜딩 전략. 스탠다드 모델(부산 기존 매장)은 현상 유지.

## 리브랜딩 배경

| 항목 | 내용 |
|------|------|
| 트리거 | 조민서 대표의 로고 리뉴얼 + 인테리어 변경 요청 |
| 판단 | 로고만 바꾸는 건 7강(비주얼)부터 시작하는 순서 실수 → 리브랜딩(리포지셔닝) 필요 |
| 깊이 | **2단계 리포지셔닝** — 이름 유지, 카테고리 내 가치 재정의 |
| 대상 | 라이트 모델 전용 (서울 미아점이 첫 사례) |

## 리브랜딩이 필요한 신호 (현재 해당)

1. **타깃 변화** — 부산 로컬 → 전국 가맹 확장, 소형 카운터석 매장
2. **카테고리 변화** — 쌀국수 단품 → 쌀국수+국밥+반미, 라이트/스탠다드 분화
3. **브랜드 확장** — 포솔 자매 브랜드 출현, 모브랜드 정체성 재정립 필요

## 불변 요소 ("바꾸지 않을 것")

- **15시간 육수** — 핵심 약속, 절대 불변
- **"라이옥" 이름** — 하노이 원조 출처 표시, 정통성 자산
- **정통성 포지셔닝** — 유지하되 표현 방식만 현대화

## 10단계 작업 체크리스트

### Phase 1: 기반 설계

**Step 1. 브랜드 정의** — 라이트의 3겹(식별/약속/의미) 정의

| 겹 | 질문 | 초안 |
|----|------|------|
| 식별 | 라이트를 들으면 뭐가 떠오르는가? | (미정) |
| 약속 | 고객에게 하는 구체적 기대치 | "빠르지만 진짜인 한 그릇" (후보) |
| 의미 | 라이트에서 먹는 것 = 어떤 자기표현? | "바쁘지만 대충 먹지 않는 사람" (후보) |

**Step 2. 포지셔닝** — 수도권 소형 쌀국수 시장의 멘탈 슬롯

포지셔닝 공식: `[타깃]에게 라이옥 라이트는 [카테고리] 중 [차별점]인 유일한 브랜드`

→ 클라이언트와 합의 필요

**Step 3. 아이덴티티 vs 이미지** — 캐퍼러 프리즘 6면 (라이트용)

| 면 | 스탠다드 | 라이트 (작업 필요) |
|----|---------|----------------|
| 체격 | 베트남풍 풀 인테리어 | 카운터석, ㄱ자 배치, 미니멀 |
| 개성 | 과묵한 장인 | (정의 필요) |
| 문화 | 하노이 50년 전통 | 동일 유지 |
| 관계 | 현지 맛 가이드 | (정의 필요) |
| 반영 | 미식가 | (정의 필요) |
| 자아상 | "진짜를 먹는 사람" | (정의 필요) |

**Step 4. 타깃 오디언스** — JTBD 기반

| 층 | 스탠다드 타깃 | 라이트 타깃 (작업 필요) |
|----|------------|---------------------|
| 인구통계 | 부산 30~40대 | 수도권 직장인 (점심/저녁) |
| 사이코그래픽 | 현지 맛 추구 미식가 | (정의 필요) |
| JTBD | 베트남 여행 기억 재현 | (정의 필요 — 예: 빠르지만 만족스러운 한 끼) |

### Phase 2: 표현 체계

**Step 5. 브랜드 스토리** — StoryBrand 7단계 라이트 버전

| 요소 | 내용 (작업 필요) |
|------|---------------|
| 영웅(고객) | 수도권 직장인, 제대로 된 점심이 그립다 |
| 외적 문제 | 빠르면서 맛있는 쌀국수를 찾기 어렵다 |
| 내적 문제 | 프랜차이즈는 다 비슷하다는 피로감 |
| 가이드(라이옥) | 하노이 50년 전통의 육수 |
| 계획 | (주문 → 빠른 서빙 → 취향 커스텀) |
| 성공 | "여기는 진짜다" 만족 |
| 실패 | 또 대충 한 끼 때우기 |

내러티브 후보: "진짜 맛에는 시간이 필요하지만, 당신의 시간은 아끼겠습니다"

**Step 6. 언어 체계**

- 라이트 전용 슬로건 (7음절 이내, 작업 필요)
- 톤 앤 보이스: 정통적이면서 따뜻한 (스탠다드와 동일 Voice, 상황별 Tone 조절)
- 금지어/필수어 목록 (작업 필요)

**Step 7. 비주얼 아이덴티티**

| 요소 | 작업 내용 |
|------|----------|
| 로고 | 스탠다드 계열 유지하되 라이트 변형 검토 |
| 주색 | 스탠다드(#0e763c)와의 관계 정의 |
| 보조색 | 최대 2개 |
| 타이포 | 제목/본문 폰트 2개 |
| 촬영 가이드 | 카운터석 환경 맞춤 |
| 공간 디자인 | **핵심** — 카운터석, ㄱ자 배치, 벽면 소재, 조명 색온도, 소품 구체화 |

### Phase 3: 경험 설계

**Step 8. 브랜드 경험** — 라이트 3대 접점

| 접점 | 설계 포인트 |
|------|-----------|
| 첫 발견 (ZMOT) | 네이버 플레이스, 배달앱, 인스타 — 3초 안에 "진짜" 전달 |
| 첫 한 입 (SMOT) | 카운터에서 빠른 서빙, 육수 온도 유지, 플레이팅 |
| 마지막 인상 (End) | 고회전 환경에서도 가능한 퇴장 설계 |

프랜차이즈 일관성 3방법:
1. 레시피 시스템화 (본사 육수 중앙공급)
2. 서비스 의식(Ritual) 표준화
3. 미스터리 쇼퍼 + 피드백 루프

### Phase 4: 진화

**Step 9. 전환 실행**

| 순서 | 내용 |
|------|------|
| 1 | 미아점 적용 (파일럿) |
| 2 | 신규 라이트 가맹점 적용 |
| 3 | 가맹점주 교육 + 서비스 매뉴얼 |

**Step 10. 콘텐츠 브랜딩**

- 리브랜딩 과정 자체를 콘텐츠로 (과정을 보여줘라)
- LP에 "왜 변화하는가" 스토리 반영
- 창업자 퍼스널 브랜딩 활용
