# 가맹 조건

쌀국수 프랜차이즈 브랜드별 가맹 조건 비교와 투명성 전략.

|--------|-----------|--------|----------|----------|
| [[포메인]] | 비공개 | 비공개 | 비공개 | ~100 |
| [[미분당]] | 1억 92만 | 1,000만 | 13평 | ~171 |

| [[미스사이공]] | ~9,600만 | 비공개 | 비공개 | ~90 |
| 월미당 | 1억 300만 | 비공개 | 15평 | ~43 |

| 헬로베트남 | 770만 | 비공개 | 비공개 | ~11 |

**업계 평균**: 외식 프랜차이즈 평균 창업비용 약 9,600만원, 투자 회수 기간 3.6년

## 비용 구조 (프랜차이즈 일반)

| 항목 | 평균 비용 | 비중 |
|------|----------|------|
| 가맹비 | 1,137만원 | 8.6% |
| 보증금 | 480만원 | 3.6% |
| 교육비 | 338만원 | 2.6% |
| 인테리어/설비 | 1억 1,286만원 | 85.2% |
| **합계** | **1억 3,242만원** | 100% |

→ 인테리어가 전체의 85%. 소형 매장으로 절감 가능한 핵심 항목.

## 핵심 발견: 투명성 부재

- 대부분 브랜드가 **가맹비, 로열티, 상세 비용을 웹상에 공개하지 않음**
- 상담/설명회 유도 전략 → 예비 창업자 불신 초래
- [[미분당]]만 마이프차에 비교적 상세한 비용 공개
- 월미당은 "순수익률 40%, 회수 3개월" 등 공격적 수치 제시 (근거 불명확)

## 차별화 기회: 투명성 전략

> "정보를 숨기지 않는 프랜차이즈"

1. **투자비 범위 공개**: "8,000만~1.2억원 (매장 규모에 따라 상이)"
2. **정보공개서 데이터 시각화**: 가맹점 평균 매출, 폐점률을 랜딩페이지에 표시
3. **비용 구성 분해**: 가맹비/교육비/인테리어비 각각 표시 (숨기지 않음)
4. **투자 회수 시뮬레이션**: 입지/규모별 예상 회수 기간 계산기 제공

| 항목 | 라이트 (10~15평) | 스탠다드 (25평+) |
|------|---------------|---------------|
| 가맹금 | 500만 | 1,000만 |
| 교육비 | 300만 | 500만 |
| 보증금 | 면제(한시적) | 면제(한시적) |
| 인테리어 | 200만/평 | 200만/평 |
| 주방설비 | 1,000~1,200만 | 1,800~2,000만 |
| 홍보·비품 | 200만 | 300만 |

### 예상 수익률 (배달 제외, 원재료비 38% 고정)

| 규모 | 월 매출 | 영업이익 | 이익률 |
|------|--------|---------|--------|
| 10~15평 | 2,000만 | 680만 | 34% |
| 25평 | 4,000만 | 1,280만 | 32% |
| 30평 | 6,000만 | 1,920만 | 32% |

### 자매 브랜드 포솔 비교 (참고)

| 항목 | 라이옥 라이트 | 포솔 (10평) |
|------|-------------|------------|
| 가맹금 | 500만 | 500만 (한정할인) |
| 총 창업비용 | ~3,200~4,200만 | 5,300만 |
| 원가율 | 38% | 35% |
| 마진율 | 34% | 43~45% |

→ 포솔은 간판/가구 별도 항목이 있어 합계가 높으나, 원가율이 낮아 운영 마진이 더 높음.

### 투명성 전략 업데이트

기존 상태("비용 일체 미공개")에서 **브로슈어를 통해 내부 데이터 확보** 완료. LP에 투명성 전략을 적용할 실 데이터가 갖추어짐:
- 가맹비/교육비/인테리어/로열티 구체 수치
- 매출 구간별 예상 영업이익률
- 2개 타입 모델 비교

## 미팅에서 확인 필요

- ~~B1: 초기 투자비~~ → 브로슈어로 확인 완료 (라이트/스탠다드)
- ~~B2: 로열티 구조~~ → 월 30만원 정액 확인
- ~~B3: 소형 매장 모델~~ → 라이트(10~15평) 확인 + 포솔 자매 브랜드
- D5: 재무 데이터 LP 공개 범위 → 수익률 데이터 있으나 **공개 허용 범위** 미확인

## 가맹사업법 주의사항

- 수익 보장/확정 매출 표현은 가맹사업법 위반 가능
- "평균 매출 OO만원" 표시 시 정보공개서 기준임을 명시해야 함
- 정보공개서 14일 전 제공 의무
