# 라이옥 라이트 비주얼 가이드라인

브랜딩 커리큘럼 7강 적용. 비주얼은 전략의 **출력물**이다.

입력: 포지셔닝("정통+밀착"), Voice("깊이·절제·자부심"), 내러티브("진짜는 빠를 수도 있다")
→ 출력: 이것을 눈에 보이게 번역.

---

## 1. CI 간극 현황과 방향 결정

### 1.1 현재 3개의 색상이 공존

| 출처 | 주 색상 | 실제 |
|------|--------|------|
| **브로슈어 CI** | 딥 그린 #0e763c | 문서상 공식 색상이나, 미아점에서 거의 안 보임 |
| **미아점 실제** | 워밍 우드 브라운 + 검정 + 금색 | 간판·인테리어·가구 전체에서 일관 |
| **포솔 CI** | 딥 그린 #00624a + 오렌지 #e5661e | 자매 브랜드, 별도 |

### 1.2 권고: 매장 현실에 CI를 맞춘다

| 옵션 | 내용 | 판단 |
|------|------|------|
| A. CI에 매장을 맞춤 | 미아점 인테리어를 딥그린으로 변경 | ❌ 비용 높음, 현재 인테리어 완성도 이미 좋음 |
| B. 매장에 CI를 맞춤 | 공식 CI를 우드+검정+금색으로 재정립 | ✅ 비용 낮음, 현실 반영, 고객 경험과 일치 |
| C. 하이브리드 | 그린을 포인트로 남기고, 우드+검정을 주력 | ⚠️ 절충안이나 색상이 많아질 위험 |

**B안 권고.** 이유:
- 미아점 인테리어가 이미 "깊이·절제·자부심" Voice와 잘 맞음
- 우드 브라운은 "전통, 자연, 따뜻함"을 전달 — 정통 베트남 포지셔닝에 적합
- 고객이 실제로 보는 것(매장)과 공식 CI가 일치해야 간극 해소
- 딥그린은 포솔과도 겹침 → 분리 효과

### 1.3 스탠다드와의 관계

스탠다드(부산)도 우드톤+검정 간판을 사용하고 있다면 → CI 통일 자연스러움.
스탠다드가 딥그린을 적극 사용 중이라면 → 라이트만 색상 변경, "같은 뿌리 다른 표현" 프레임.

→ **클라이언트 확인 필요**: 부산 스탠다드 매장의 실제 간판/인테리어 색상

## 2. 색상 체계 (Color System)

### 2.1 라이트 전용 색상 (미아점 기반 추출)

| 역할 | 색상 | 근사치 코드 | 사용 |
|------|------|-----------|------|
| **주 색상** | 워밍 우드 브라운 | #8B6914 ~ #A0784C | 카운터, 칸막이, 가구, 입간판 프레임 |
| **구조 색상** | 다크/블랙 | #1A1A1A ~ #2D2D2D | 간판 배경, 외부 프레임, 텍스트 |
| **악센트** | 골드/머스타드 | #C8A951 ~ #D4A017 | 간판 텍스트, 스툴 쿠션, 포인트 |
| **중성** | 크림/베이지 | #F5F0E1 ~ #E8DCC8 | 벽면, 바닥, 배경 |

### 2.2 "왜 이 색인가" (7강 핵심)

| 색상 | 감정 | 브랜드 연결 |
|------|------|-----------|
| 우드 브라운 | 전통, 자연, 따뜻함 | "하노이 50년 전통" — 시간이 깃든 나무의 색 |
| 블랙 | 절제, 자신감, 프리미엄 | "떠들지 않아도 맛이 증명" — Voice의 절제 |
| 골드 | 깊이, 가치, 품격 | "15시간의 깊이" — 육수의 색이기도 함 |
| 크림 | 편안함, 깨끗함 | 음식이 돋보이는 배경 |

### 2.3 금지 색상

| 색상 | 이유 |
|------|------|
| 딥 그린 (#0e763c) | 포솔 CI와 혼동, 미아점에서 미사용 |
| 비비드 레드/오렌지 | "할인", "저렴한" 연상. Voice의 절제와 충돌 |
| 파스텔 톤 | "감성 카페" 느낌. 미분당 영역과 겹침 |

## 3. 로고

### 3.1 현재 상태 (미아점 사진 기반)

- 한글 "라이옥" — 금색/머스타드, 깔끔한 고딕 계열
- 하단 서브텍스트: "베트남 쌀국수" (작은 글씨)
- 브로슈어 로고(Phở Lai Ngọc + 한자 심볼)와는 다른 느낌

### 3.2 권고

| 요소 | 방향 |
|------|------|
| 코어 로고 | 한글 "라이옥" 유지 — 간결, 가독성 높음 |
| 서브마크 | "LIGHT" 또는 "라이트" 서브텍스트 추가 → 스탠다드와 구분 |
| 베트남어 | "Phở Lai Ngọc" 소형으로 병기 가능 → 정통성 시각적 신호 |
| 스타일 | 세리프 또는 세리프+고딕 혼합 — 전통+현대 |

### 3.3 사용 규칙

| 상황 | 로고 버전 |
|------|---------|
| 간판 (검정 배경) | 골드 텍스트 "라이옥" + 하단 "LIGHT" |
| 간판 (우드 배경) | 다크 텍스트 "라이옥" + 하단 "LIGHT" |
| 메뉴판/인쇄물 | 다크 버전 (밝은 배경용) |
| SNS 프로필 | 정사각형 심볼 버전 (축소 가독성) |
| 최소 여백 | 로고 높이의 50% 이상 사방 확보 |

## 4. 타이포그래피

### 4.1 2폰트 시스템

| 역할 | 서체 방향 | 느낌 |
|------|---------|------|
| **제목/간판** | 세리프 계열 (명조) | 전통, 권위, 깊이 — "15시간"의 무게감 |
| **본문/설명** | 산세리프 계열 (고딕) | 깔끔, 가독성, 현대적 — "빠르게"의 효율 |

### 4.2 후보 (한글 기준)

| 역할 | 후보 서체 | 이유 |
|------|---------|------|
| 제목 | Noto Serif KR / 나눔명조 / Pretendard (Bold) | 웹+인쇄 모두 사용 가능 |
| 본문 | Pretendard / Noto Sans KR | 가독성 + 다양한 웨이트 |

→ 최종 선정은 디자이너와 협의. **핵심 원칙: 최대 2개, 웨이트로 변화.**

## 5. 이미지 스타일 (촬영 가이드)

### 5.1 3줄 촬영 규칙

1. **톤**: 따뜻한 자연광 or 따뜻한 인공조명. 차갑거나 형광등 느낌 금지.
2. **앵글**: 카운터 위에 놓인 한 그릇 + 김이 올라오는 장면. 45도 또는 정면.
3. **배경**: 우드 카운터 또는 베트남 문양 벽면이 자연스럽게 포함. 흰 배경 스튜디오 사진 지양.

### 5.2 필수 컷 (기본 세트)

| # | 컷 | 용도 | 포인트 |
|---|-----|------|--------|
| 1 | 카운터 위 소고기 쌀국수 정면 (김 올라옴) | 대표 이미지, LP 히어로 | 그릇 뒤로 카운터+주방 흐릿하게 |
| 2 | 위에서 내려다본 한 그릇 (플랫레이) | SNS, 배달앱 | 고수·숙주·소스 세팅 포함 |
| 3 | 면을 젓가락으로 들어올리는 장면 | 식욕 자극, 광고 | 육수가 흘러내리는 순간 |
| 4 | 카운터에서 조리하는 과정 (직원 뒷모습) | 스토리, LP | "눈앞에서" 경험 전달 |
| 5 | 카운터에 앉은 고객의 옆모습 (자연스러운) | LP, SNS | 혼밥의 편안함 |
| 6 | 매장 전체 분위기 (입구에서 안을 바라보는) | LP, 네이버 플레이스 | 우드톤+조명+카운터 전체감 |
| 7 | 육수 냄비/솥 클로즈업 | 스토리 | "15시간"의 시각적 증거 |

### 5.3 금지 이미지

| 금지 | 이유 |
|------|------|
| 흰 배경 스튜디오 촬영 | 매장 경험과 분리됨 |
| 과도한 보정/필터 | Voice의 "절제"에 위배 |
| 음식 없이 매장만 찍은 사진 | 핵심은 "한 그릇" |
| 포장 용기 사진 위주 | 라이트의 핵심은 매장 경험 |

## 6. 공간 디자인 가이드 (인테리어)

### 6.1 미아점에서 유지할 것 (이미 잘 되어 있는 것)

| 요소 | 현재 | 유지 이유 |
|------|------|----------|
| 우드 카운터 | 따뜻한 우드톤, 긴 형태 | 핵심 경험 구조. 조리 과정 관람 |
| 목재 세로 바 칸막이 | 공간 분리 + 시각적 리듬 | 소형이지만 밀집감 완화 |
| 베트남 기하학적 문양 벽면 | 금색/갈색 톤 | 정통성의 시각적 신호. 사진 배경으로도 우수 |
| 한지 둥근 펜던트 조명 | 따뜻한 빛 | 아시안 감성 + 음식 색감 보정 효과 |
| 머스타드 스툴 쿠션 | 카운터석 포인트 | 악센트 컬러와 일치 |

### 6.2 강화할 것

| 요소 | 현재 | 개선 방향 |
|------|------|----------|
| **"15시간" 사이니지** | 없음 | 카운터 뒤 벽면 또는 메뉴판 상단에 **"15시간."** 한 마디. Voice의 절제를 공간으로 번역 |
| **조리 과정 가시성** | 오픈 구조이나 의도적 연출 없음 | 카운터에서 육수 냄비/면 건지기가 자연스럽게 보이도록 주방 동선 조정 |
| **브랜드 로고 통일** | 간판마다 스타일 다름 (사진 2 vs 7) | 라이트 전용 간판 규격 확정 |
| **향기 설계** | 자연 발생 | 환기 방향을 카운터 쪽으로 → 고객에게 향신료 향 자연 전달 |

### 6.3 신규 가맹점 인테리어 체크리스트

| # | 항목 | 사양 |
|---|------|------|
| 1 | 카운터 | 우드톤, 길이 최소 3m, 주방과 밀착 |
| 2 | 스툴 | 머스타드/골드 톤 쿠션, 등받이 없는 형태 |
| 3 | 칸막이 | 세로 목재 바, 높이 1.8m 이상 |
| 4 | 벽면 | 베트남 기하학적 문양 패널 최소 1면 |
| 5 | 조명 | 한지 펜던트 2개 이상 + 카운터 하단 간접조명 |
| 6 | 바닥 | 기하학 패턴 타일 또는 베이지 타일 |
| 7 | 간판 | 라이트 규격 (검정 or 우드 배경 + 골드 "라이옥 LIGHT") |
| 8 | 사이니지 | "15시간." 벽면 사이니지 1개소 이상 |
| 9 | 키오스크 | 입구 또는 카운터 끝에 배치 |
| 10 | 소수 테이블 | 원형 2인 테이블 1~2개 (공간 여유 시) |

## 9. 클라이언트 확인 필요 사항

1. **스탠다드 매장의 실제 간판/인테리어 색상** — 딥그린 사용 중인지, 우드톤인지
2. **CI를 매장에 맞추는 것에 동의하는지** — 딥그린 폐기/축소 의사
3. **"라이옥 LIGHT" 서브마크 사용** 동의 여부
4. **인테리어 가이드라인 구속력** — 신규 가맹점 필수 vs 권장
5. **촬영 진행 여부** — 프로 촬영 필요 시 일정/예산
