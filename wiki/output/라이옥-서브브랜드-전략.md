# 라이옥 서브브랜드 네이밍+색상+컨셉 전략

"라이옥 라이트"라는 이름이 최선인지, 색상·컨셉과 통합된 서브브랜드 전략을 재설계.

---

## 1. "라이트"의 문제점

| 문제 | 설명 |
|------|------|
| **"가벼운"의 뉘앙스** | "Light beer" = 원본보다 덜한 것. "라이트 쌀국수" = 스탠다드의 축소판? |
| **포지셔닝과 충돌** | "깊이·절제·자부심"을 소구하면서 이름이 "라이트"이면 모순 |
| **"타협 없는"과 모순** | "타협 없는 맛"인데 모델명이 "라이트(가벼운)"이면 신뢰 약화 |
| **영어 직역 문제** | "Light"는 한국에서 "다이어트/저칼로리/간편" 연상. 정통성 훼손 |

포메인 RED와 비교:
- RED = Rice noodles. Express. Delivery → 의미가 이중 작동 (색상+약어)
- LIGHT = 단순히 "가벼운/소형" → 단일 의미, 부정적 뉘앙스

---

## 2. 서브브랜드 전략의 두 가지 방향

### 방향 A: 서브브랜드 신설 (포메인 RED처럼)

"라이옥 ○○○"으로 별도 서브 네임. 색상+컨셉을 담은 이름.

- 장점: 스탠다드와 명확 구분, 독자적 아이덴티티
- 단점: 브랜드 복잡성 증가, 고객이 "라이옥인데 다른 거?"로 혼란 가능

### 방향 B: 서브브랜드 없이 매장 포맷만 구분

모든 매장이 "라이옥". 소형 매장은 별도 이름 없이 인테리어/메뉴로 자연스럽게 차이. (스타벅스 모델 — 리저브/일반 구분은 있지만 다 "스타벅스")

- 장점: 브랜드 단순, "라이옥"이라는 이름의 힘 집중
- 단점: 가맹 모집 시 모델 구분이 불명확, LP에서 소형 모델 별도 소구 어려움

### 판단

라이옥의 경우 **방향 A**가 적합. 이유:
- 스탠다드(25평+ 테이블) vs 소형(10~15평 카운터)은 경험이 본질적으로 다름
- 가맹 모집 시 "라이트 모델"을 별도로 소구해야 함 (비용·타깃 다름)
- 단, 이름이 경험의 가치를 **올려야** 함. "라이트"처럼 내려서는 안 됨

## 3. 서브브랜드 후보 5안

### 안 1: 라이옥 골드 (GOLD)

| 항목 | 내용 |
|------|------|
| **이름** | 라이옥 GOLD |
| **색상 연결** | 15시간 육수의 색 = 깊은 금빛 |
| **의미** | "정수", "가장 깊은 것", "프리미엄" |
| **컨셉** | 소형이지만 가장 진한 경험. "작지만 골드" |
| **색상 체계** | 주: 골드/머스타드 #C8A951, 구조: 블랙, 보조: 우드 브라운 |
| **포지셔닝 적합** | ★★★★☆ — "깊이"와 연결되지만 "프리미엄"이 가격대(₩11,000)와 맞는지 |
| **위험** | "골드"가 과하면 기대치가 올라감. 소형 매장에서 "골드"가 과장으로 느껴질 수 있음 |

소비자 카피: "15시간 육수의 골드, 눈앞에서"
가맹 카피: "골드 한 그릇의 가치를, 합리적인 비용으로"

### 안 2: 라이옥 앰버 (AMBER)

| 항목 | 내용 |
|------|------|
| **이름** | 라이옥 AMBER |
| **색상 연결** | 호박색 = 육수의 실제 색, 따뜻한 우드톤과 일치 |
| **의미** | 호박(amber)은 **시간이 만든 보석**. 15시간 육수 = 시간이 만든 맛 |
| **컨셉** | "시간이 깃든 한 그릇" |
| **색상 체계** | 주: 앰버/호박 #D4A017, 구조: 다크 우드, 보조: 크림 |
| **포지셔닝 적합** | ★★★★★ — 시간(15시간)+색상(육수)+공간(우드톤) 모두 연결 |
| **위험** | "앰버"가 한국 소비자에게 생소. "무슨 뜻이지?" 한 번 더 생각해야 함 |

소비자 카피: "시간이 만든 색, 앰버"
가맹 카피: "15시간의 깊이를, 당신의 매장으로"

### 안 3: 라이옥 원 (ONE)

| 항목 | 내용 |
|------|------|
| **이름** | 라이옥 ONE |
| **색상 연결** | 독립적 — 색상보다 컨셉 중심 |
| **의미** | 1인, 한 그릇, 하나에 집중, "이것 하나면 된다" |
| **컨셉** | "한 그릇의 하노이" — 기존 슬로건과 직결 |
| **색상 체계** | 기존 우드+블랙+골드 유지 (색상 변화 불필요) |
| **포지셔닝 적합** | ★★★☆☆ — 미니멀하고 깔끔하지만 "깊이"보다 "간결"에 치우침 |
| **위험** | "ONE"이 너무 추상적. 1인 매장? 1인분? 의미 분산 |

소비자 카피: "한 그릇이면 됩니다"
가맹 카피: "하나의 매장, 하나의 집중"

### 안 4: 라이옥 바 (BAR / BÀR)

| 항목 | 내용 |
|------|------|
| **이름** | 라이옥 BAR (또는 베트남어 BÀR) |
| **색상 연결** | 독립적 — 공간(카운터/바) 직접 표현 |
| **의미** | 카운터에서 즐기는 경험. 일본 라멘바, 스시바처럼 "바=전문성" |
| **컨셉** | "카운터에서 즐기는 정통 한 그릇" |
| **색상 체계** | 우드+블랙 강조 (바/카운터 느낌) |
| **포지셔닝 적합** | ★★★★☆ — 공간 경험을 네이밍에 담음. "밀착" 포지셔닝 직결 |
| **위험** | "바"가 술집/주점 연상. 쌀국수+바의 조합이 어색할 수 있음 |

소비자 카피: "카운터에서 만나는 하노이"
가맹 카피: "바 한 칸의 가치, 라이옥이 증명합니다"

### 안 5: 라이옥 15 (FIFTEEN)

| 항목 | 내용 |
|------|------|
| **이름** | 라이옥 15 |
| **색상 연결** | 독립적 — 숫자가 곧 브랜드 |
| **의미** | 15시간 육수. 15평 매장. 숫자 하나에 모든 것이 담김 |
| **컨셉** | "15시간이 만든 한 그릇, 15평의 집중 공간" |
| **색상 체계** | 기존 우드+블랙+골드 유지 + "15" 타이포가 핵심 비주얼 |
| **포지셔닝 적합** | ★★★★★ — 가장 기억에 남음. "15"라는 숫자가 곧 차별점 |
| **위험** | "15"만으로는 의미 전달이 즉각적이지 않음. 설명이 한 번 필요 ("15시간 육수요") |

소비자 카피: "15시간의 깊이, 15평의 집중"
가맹 카피: "15. 이 숫자가 라이옥의 전부입니다"

## 5. 최종 권고: 안 5 "라이옥 15" + 안 2 "앰버" 색상 결합

### 왜 이 조합인가

**"15"가 이름**, **앰버가 색상**:

- "라이옥 15" — 숫자가 곧 브랜드. 15시간 육수 + 15평 공간. 설명이 필요하지만, **한번 들으면 절대 잊지 않음**
- 앰버 색상 — 15시간 육수의 실제 색. "시간이 만든 보석" = 호박(amber). 우드톤 인테리어와 자연 일치
- Voice "절제"에 가장 부합 — 이름이 숫자 하나. 설명 없이 "15." 그게 다.

### 통합 아이덴티티

```
┌─────────────────────────────────────────────┐
│                                             │
│  이름: 라이옥 15                             │
│  내러티브: "진짜는 빠를 수도 있다"            │
│  Voice: 깊이 · 절제 · 자부심                 │
│                                             │
│  [색상]                                     │
│  ■ 앰버 #D4A017 (주)     ■ 블랙 (구조)      │
│  ■ 우드 브라운 (보조)     □ 크림 (중성)      │
│                                             │
│  [슬로건]                                   │
│  소비자: "15시간의 깊이, 눈앞에서"           │
│  가맹: "15. 이 숫자 하나로 시작하세요"       │
│                                             │
│  [매장 사이니지]                             │
│  벽면에 "15" 대형 타이포. 그 아래 작게       │
│  "시간의 깊이를, 가까이서"                   │
│                                             │
│  [스탠다드와의 관계]                         │
│  스탠다드 = "라이옥" (그린 or 우드, 기존)    │
│  소형 = "라이옥 15" (앰버+블랙)              │
│  → "같은 15시간, 다른 공간"                  │
│                                             │
└─────────────────────────────────────────────┘
```

### "15"가 스탠다드에도 적용될 수 있는가?

"15시간 육수"는 스탠다드도 마찬가지. 차이점:

| | 스탠다드 | 라이옥 15 |
|--|---------|---------|
| 15시간 육수 | ✅ | ✅ |
| 15평 매장 | ❌ (25평+) | ✅ |
| "15"의 의미 | 육수 시간 | **육수 + 공간 이중 의미** |

→ "15"라는 숫자가 소형 모델에서만 이중 작동하므로, 서브브랜드로서 차별성 확보.

## 6. 스탠다드의 색상은?

라이옥 15가 앰버+블랙이면, 스탠다드는?

| 옵션 | 스탠다드 색상 | 판단 |
|------|------------|------|
| A | 기존 딥그린 유지 | 브로슈어와 일치. 라이옥 15와 색상으로 명확 구분 |
| B | 우드 브라운 (현재 매장) | 실제와 일치하나 라이옥 15와 겹침 |
| C | 새로운 색상 도입 | 비용 큼, 혼란 |

**권고: A안** — 스탠다드는 딥그린, 라이옥 15는 앰버+블랙.
이렇게 하면:
- 딥그린 = 전통, 정통, 풍성한 경험 (스탠다드)
- 앰버 = 시간, 깊이, 집중된 경험 (15)
- 두 색상 모두 "자연/전통" 계열이라 모 브랜드 아래 통일감 유지

## 7. 포메인 RED와의 전략 비교

| 요소 | 포메인 RED | 라이옥 15 (권고안) |
|------|----------|-----------------|
| 약어 | Rice.Express.Delivery | 15시간 육수, 15평 공간 |
| 색상 | 빨강 (에너지, 속도) | 앰버 (시간, 깊이, 따뜻함) |
| 모델 | 배달·포장 전용 | 카운터+소수 테이블 홀 |
| 뉘앙스 | "빠르고 효율적" | "깊지만 집중된" |
| 부모 브랜드 | 포메인 (19년 원조) | 라이옥 (10년 정통) |

→ 포메인 RED가 "속도"를 택했다면, 라이옥 15는 **"깊이"**를 택함. 같은 소형이지만 가치의 방향이 다름.
