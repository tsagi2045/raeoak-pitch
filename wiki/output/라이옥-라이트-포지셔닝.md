# 라이옥 라이트 포지셔닝 확정

브랜딩 커리큘럼 2강 프레임워크 적용. 라이트 모델의 포지셔닝을 정의한다.

---

## 1. 포지셔닝 공식

> **[타깃]에게 [브랜드]는 [카테고리] 중 [핵심 차별점]인 유일한 브랜드다.**

좋은 포지셔닝의 3조건:
1. 고객이 실제로 원하는 것
2. 경쟁자가 쉽게 따라할 수 없는 것
3. 한 문장으로 말할 수 있는 것

---

## 2. 수도권 쌀국수 멘탈 슬롯 (현재)

고객 머릿속에 "쌀국수" 하면 떠오르는 2~3개 자리:

| 슬롯 | 점유 브랜드 | 키워드 |
|------|-----------|--------|
| "대중적 체인" | 에머이, 미분당 | 접근성, 감성, 어디든 있음 |
| "원조/프리미엄" | 포메인 | 19년, 대형 매장, 가격↑ |
| "가성비 동네 맛집" | 로컬 개인점 | 저렴, 단골, 불안정 |
| "현지식 급성장" | 호앙비엣 | 현지 공장, 수도권 확장 |
| **"정통+밀착"** | **비어 있음** | ? |

**핵심**: 수도권에서 "정통 베트남 육수 + 소형 밀착 경험"을 소구하는 프랜차이즈가 없다.

- 에머이/미분당: 대중적이지만 정통성 약함
- 포메인: 정통적이지만 대형 매장, 가격 높음
- 호앙비엣: 현지식이지만 공간 경험에 대한 이야기 없음
- 로컬 맛집: 맛은 있을 수 있지만 프랜차이즈 시스템 없음

---

## 3. 라이트만 쓸 수 있는 차별점

| 차별점 후보 | 고객이 원하는가? | 따라할 수 없는가? | 한 문장 가능? |
|-----------|:---:|:---:|:---:|
| 15시간 육수 | ✅ 깊은 맛 | ✅ 본사 중앙공급 체계 | ✅ |
| 카운터에서 조리가 눈앞에 | ✅ 라이브 경험 | ✅ 소형 매장 구조에서만 가능 | ✅ |
| 베트남인 80%가 인정 | ✅ 사회적 증거 | ✅ 타 브랜드 주장 불가 | ✅ |
| 빠른 회전 + 정통 품질 | ✅ 직장인 점심 니즈 | ⚠️ 일부 경쟁 가능 | ✅ |
| 소자본 5천만 이하 | ✅ 창업자 니즈 | ⚠️ 경쟁 가능 | ✅ |

**가장 강력한 조합**: 15시간 육수(따라할 수 없는 품질) + 밀착 경험(소형 매장 구조적 특성) + 베트남인 80%(사회적 증거)

---

## 4. 포지셔닝 후보 3안

### 안 A: "밀착 정통"

> **바쁜 수도권 직장인에게, 라이옥 라이트는 쌀국수 전문점 중 15시간 정통 육수를 눈앞에서 경험하는 유일한 브랜드다.**

| 항목 | 평가 |
|------|------|
| 핵심 단어 | "눈앞에서" |
| 강점 | 카운터 경험을 차별점으로 명확히 설정. 공간과 맛을 동시에 소구 |
| 약점 | "눈앞에서"가 모든 오픈키친 매장에 적용 가능할 수 있음 |
| 한 단어 요약 | **밀착** |

소비자 한 줄: "15시간 육수, 눈앞에서"
가맹 한 줄: "정통 육수의 밀착 경험을 소자본으로"

### 안 B: "진짜의 가까운 버전"

> **제대로 된 한 끼가 그리운 직장인에게, 라이옥 라이트는 쌀국수 전문점 중 부산 10년 검증된 정통 육수를 일상에서 만나는 유일한 브랜드다.**

| 항목 | 평가 |
|------|------|
| 핵심 단어 | "일상에서 만나는" |
| 강점 | 부산 본점의 브랜드 자산을 계승. "원래는 멀리 가야 먹을 수 있는 맛"이라는 희소성 |
| 약점 | 공간 경험(카운터)이 빠져 있음. 수도권 고객에게 "부산"이 의미 있을지 미지수 |
| 한 단어 요약 | **일상** |

소비자 한 줄: "부산 줄 서던 그 맛, 일상에서"
가맹 한 줄: "10년 검증된 맛, 가까운 일상으로"

### 안 C: "빠르지만 진짜"

> **빠르면서도 진짜인 쌀국수를 원하는 사람에게, 라이옥 라이트는 쌀국수 전문점 중 15시간 육수의 깊이를 타협 없이 빠르게 전하는 유일한 브랜드다.**

| 항목 | 평가 |
|------|------|
| 핵심 단어 | "타협 없이 빠르게" |
| 강점 | 속도와 품질의 양립이라는 보편적 니즈 직격. 기존 "작지만 진짜" 포지셔닝의 자연스러운 확장 |
| 약점 | "빠른 쌀국수"는 배달앱과 혼동될 수 있음. 공간 경험이 약함 |
| 한 단어 요약 | **타협 없는** |

소비자 한 줄: "15시간 걸리는 맛, 당신은 10분"
가맹 한 줄: "타협 없는 맛, 합리적인 창업"

---

## 5. 3안 비교

| 기준 | 안 A (밀착) | 안 B (일상) | 안 C (타협없는) |
|------|:---------:|:---------:|:------------:|
| 고객이 원하는가 | ★★★ | ★★☆ | ★★★ |
| 따라할 수 없는가 | ★★★ | ★★☆ | ★★☆ |
| 한 문장 가능 | ★★★ | ★★★ | ★★★ |
| 기존 포지셔닝 계승 | ★★☆ | ★★★ | ★★★ |
| 카운터 경험 반영 | ★★★ | ★☆☆ | ★☆☆ |
| 가맹 모집에도 작동 | ★★☆ | ★★☆ | ★★★ |
| LP 비주얼 전개 | ★★★ | ★★☆ | ★★☆ |

---

## 6. 권고: A안 + C안 결합

단독으로는 A안("밀착")이 가장 차별적이고, C안("타협 없는")이 가장 설득력 있다. 결합하면:

> **바쁘지만 타협하지 않는 사람에게, 라이옥 라이트는 쌀국수 전문점 중 15시간 정통 육수를 눈앞에서 빠르게 경험하는 유일한 브랜드다.**

### 한 단어: **"타협 없는 밀착"**

### 이 포지셔닝이 결정하는 것들

| 영역 | 방향 |
|------|------|
| 슬로건 | "15시간의 깊이, 눈앞에서" / "타협 없는 한 그릇" 계열 |
| 인테리어 | 카운터에서 조리 과정이 보이는 구조 유지·강화 |
| 메뉴 | 핵심 메뉴 집중 (다양성보다 깊이) |
| 가격 | 합리적이되 저가가 아님 (정통성에 맞는 가격대) |
| 사진 스타일 | 카운터 위에 놓인 한 그릇, 김이 올라오는 장면, 조리 과정 |
| 톤 앤 보이스 | 자신감 있고 군더더기 없는 ("대충 안 합니다") |
| LP 히어로 | "15시간 걸리는 맛, 당신은 10분이면 됩니다" |
| 가맹 메시지 | "타협 없는 맛을, 부담 없는 비용으로" |

---

## 7. 검증 질문 (클라이언트 합의 필요)

이 포지셔닝을 확정하기 전에 조민서 대표와 확인할 것:

1. **"빠른 서빙"이 라이트의 실제 운영과 맞는가?** — 카운터 매장이라 빠를 것으로 추정하지만, 실제 서빙 시간 확인 필요
2. **카운터에서 조리 과정이 실제로 보이는가?** — 사진에서 오픈 구조로 보이지만, 어느 정도 수준인지 확인
3. **"타협 없는"이 가맹점주에게 부담이 되지 않는가?** — 품질 기준이 높으면 운영이 어려울 수 있음. 본사 중앙공급으로 해소 가능한지
4. **스탠다드와의 관계를 어떻게 표현할 것인가?** — "라이옥의 라이트 버전" vs "라이옥의 새로운 경험"
