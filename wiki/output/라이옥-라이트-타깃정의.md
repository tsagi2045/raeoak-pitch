# 라이옥 라이트 타깃 정의

브랜딩 커리큘럼 4강 프레임워크 적용. 소비자(B2C)와 가맹 예비 창업자(B2B) 양쪽 정의.

---

## 1. 배제 먼저 (4강 핵심: "누구를 제외할지")

### 1.1 소비자 — 라이트에 오지 않을 사람

| 제외 대상 | 이유 | 그들이 갈 곳 |
|----------|------|-----------|
| 가격만 보는 고객 | ₩11,000은 최저가가 아님 | 리틀하노이 (₩3,900) |
| 4인+ 모임·회식 | 카운터+소수 테이블로 대규모 불가 | 스탠다드, 꾸아, 호앙비엣 |
| "아무 쌀국수나" 고객 | 정통성 체감 불가, 충성도 없음 | 편의점 컵쌀국수, 배달앱 |
| 감성 공간 목적 | 사진 찍기, 데이트 | 미분당 (감성 인테리어) |
| 배달만 하는 고객 | 카운터 밀착 경험 무의미 | 배달의쌀국수 |

### 1.2 가맹 — 라이트에 맞지 않는 창업자

| 제외 대상 | 이유 | 적합 대안 |
|----------|------|---------|
| 다점포 운영자 (비점주 상주) | 라이트는 점주 참여 필수 (1인 운영) | 스탠다드 또는 타 브랜드 |
| 대형 매장 원하는 창업자 | 라이트는 10~15평 고정 | 스탠다드 (25평+) |
| ROI 데이터만 보는 창업자 | 라이트 실적 데이터 아직 부족 (미아점 1개) | 미분당 (171개 데이터) |

## 2. 소비자 타깃 (B2C) — 3층 정의

### 2.1 1층: 인구통계

| 항목 | 라이트 핵심 타깃 |
|------|---------------|
| 연령 | 25~45세 |
| 지역 | 수도권 직장 밀집 지역 (오피스 상권) |
| 소득 | 중간 이상 (점심 ₩11,000 지출 가능) |
| 가구 | 1인 가구 또는 맞벌이 |
| 외식 빈도 | 주 3회 이상 |

### 2.2 2층: 사이코그래픽

| 특성 | 설명 |
|------|------|
| 음식 기준 | 있다. "아무거나"로 때우지 않는다 |
| 시간 가치 | 높다. 점심 40분 안에 끝내야 한다 |
| 브랜드 인식 | 프랜차이즈를 경계하되, "진짜"에는 열림 |
| 혼밥 태도 | 익숙하다. 카운터에서 혼자 먹는 것 자연스러움 |
| 건강 관심 | 있다. 기름진 음식보다 국물/면류 선호 |
| SNS | 맛있으면 올린다. 강요하지 않아도 자발적 공유 |

### 2.3 3층: 행동 맥락 — JTBD (Jobs to Be Done)

고객은 "쌀국수"를 사는 게 아니라, **특정 상황에서 특정 문제를 해결하기 위해 "고용"**한다.

#### Job 1: "빠르지만 대충은 싫어" (Primary)

> **When** 점심시간이 40분밖에 없는데,
> **I want to** 빠르면서도 맛있는 한 끼를 먹고 싶다.
> **So I can** 오후 업무에 든든하고 기분 좋게 복귀할 수 있다.

| 항목 | 내용 |
|------|------|
| 상황 | 직장인 점심, 시간 제한 |
| 대안 | 김밥천국(빠르지만 대충), 미분당(맛있지만 좀 더 시간), 편의점(빠르지만 허전) |
| 라이트의 답 | 15시간 육수 + 빠른 서빙 + 카운터 = "빠르면서 진짜" |
| 고용 빈도 | 주 2~3회 반복 가능 |

#### Job 2: "혼자 먹어도 특별한 경험" (Secondary)

> **When** 혼자 밥을 먹는데 허전하지 않은 곳이 필요할 때,
> **I want to** 편하면서도 뭔가 경험이 있는 곳에서 먹고 싶다.
> **So I can** 혼밥이지만 "잘 먹었다"는 만족감을 느낄 수 있다.

| 항목 | 내용 |
|------|------|
| 상황 | 1인 가구 저녁, 혼밥 |
| 대안 | 편의점(편하지만 허전), 배달(편하지만 혼자 먹기 쓸쓸), 일반 식당(혼자 가기 부담) |
| 라이트의 답 | 카운터에서 눈앞의 조리 과정 = 혼밥이 "관람"이 되는 경험 |
| 고용 빈도 | 주 1~2회 |

#### Job 3: "기름진 거 말고" (Tertiary)

> **When** 연속된 회식이나 기름진 식사 후,
> **I want to** 속이 편하면서도 든든한 한 끼를 먹고 싶다.
> **So I can** 몸 컨디션을 회복할 수 있다.

| 항목 | 내용 |
|------|------|
| 상황 | 해장, 컨디션 관리, 다이어트 |
| 대안 | 죽집(편하지만 맛 단조), 샐러드(건강하지만 안 든든) |
| 라이트의 답 | 맑은 육수 + 쌀면 = 속 편하면서 든든 |
| 고용 빈도 | 비정기 |

### 2.4 소비자 페르소나

> **윤서진, 32세, IT기업 PM**
>
> 강남 오피스에서 일한다. 점심은 보통 40분. 매일 "뭐 먹지"가 고민이다. 김밥천국은 질렸고, 줄 서는 맛집은 시간이 안 된다. 혼밥에 익숙하고 카운터석이 오히려 편하다. 음식에 대한 기준이 있어서 "어디가 맛있다"는 리뷰를 꼼꼼히 본다. 맛있으면 인스타 스토리에 올린다.
>
> **라이옥 라이트를 검색하기 직전에 무슨 일이 있었는가?**
> → 오늘도 점심 뭐 먹을지 고민. "맵지 않고 든든한 거" 검색.
>
> **메뉴를 고르기 전에 마지막으로 망설이는 이유는?**
> → "프랜차이즈 쌀국수는 다 비슷하지 않나?" → 15시간 육수, 베트남인 80% 보고 결심.
>
> **식사를 마친 후 누구에게 뭐라고 말하는가?**
> → "여기 육수 진짜 깊더라. 카운터에서 끓이는 거 보이는데 그것도 좋았어."

## 3. 가맹 타깃 (B2B) — 라이트 전용 재정의

### 3.1 기존 분석과의 차이

기존 [[라이옥-페르소나-설득전략]]에서:
- Primary: 탈직장인 (A)
- Secondary: 소자본 (C, ↑)
- 3순위: 다점포 (B, ↓)

**라이트 전용으로 재조정:**

| 순위 | 일반 라이옥 | 라이트 전용 | 이유 |
|------|-----------|-----------|------|
| **Primary** | 탈직장인 | **소자본 첫 창업 (↑↑)** | 라이트 총비용 3,200~4,200만 = 소자본 완벽 매칭 |
| **Secondary** | 소자본 | **탈직장인 (유지)** | 1인 운영 원하는 탈직장인도 라이트 적합 |
| **제외** | 다점포 | **다점포 (제외)** | 라이트 1인 운영 = 비점주 상주 불가 |

### 3.2 라이트 Primary: 소자본 첫 창업

| 항목 | 내용 |
|------|------|
| 연령 | 25~35세 |
| 자산 | 3,000만~7,000만원 |
| 경험 | 사업/창업 경험 없음 |
| 동기 | "내 가게를 갖고 싶다" + "대기업 대안" |
| 최대 불안 | "실패하면 어떡하지" |
| 정보 수집 | 유튜브, 마이프차, 인스타 |

#### JTBD

> **When** 직장 생활에 한계를 느끼고 창업을 생각하기 시작할 때,
> **I want to** 적은 돈으로 실패 위험이 낮은 검증된 아이템을 찾고 싶다.
> **So I can** 경제적 독립을 하면서도 자부심을 느낄 수 있다.

#### 라이트가 이 Job에 응답하는 방식

| 불안 | 라이트의 답 |
|------|-----------|
| "돈이 부족해" | 총 3,200~4,200만원, 5천만원 이하 |
| "혼자 할 수 있을까" | 1인 운영 시스템, 본사 육수 중앙공급 |
| "맛이 일정할까" | 15시간 육수 본사 제조, 라면보다 쉬운 조리 |
| "손님이 올까" | 부산 10년 검증, 베트남인 80%, 다이닝코드 4.8 |
| "본사가 도와줄까" | 13개 매장이기에 1:1 밀착 SV |

#### 핵심 메시지

> "5천만원 이하로, 10년 검증된 맛으로, 혼자서도 운영하는 내 가게. 라이옥 라이트."

#### CTA

"무료 창업 상담 신청" / "라이트 모델 상세 자료 받기"

### 3.3 라이트 Secondary: 탈직장인 (1인 운영 희망)

| 항목 | 내용 |
|------|------|
| 연령 | 35~49세 |
| 자산 | 1억~2.5억 (퇴직금) |
| 경험 | 직장 15년+, 사업 경험 없음 |
| 동기 | "할 거면 제대로" + "부담 없이 시작" |
| 특이점 | 돈은 있지만 **큰 매장 운영 부담**을 피하고 싶어 라이트 선택 |

#### JTBD

> **When** 퇴직 후 제2의 인생을 준비하는데,
> **I want to** 큰 투자 없이 혼자서 운영할 수 있는 품격 있는 가게를 하고 싶다.
> **So I can** 경제적 안정과 자부심을 동시에 얻을 수 있다.

#### 핵심 메시지

> "대형 매장의 부담 없이, 15시간 육수의 자부심은 그대로. 라이옥 라이트는 경험 많은 분을 위한 선택입니다."

## 4. 소비자 vs 가맹 타깃 연결

| | 소비자 (먹는 사람) | 가맹 (여는 사람) |
|--|-----------------|----------------|
| Primary | 바쁜 직장인 (빠르지만 진짜) | 소자본 첫 창업 (5천만 이하) |
| Secondary | 혼밥족 (카운터 경험) | 탈직장인 (1인 부담 없이) |
| Tertiary | 건강식 선호 | — |

**연결 고리**: 소비자가 라이트에서 좋은 경험 → "나도 이런 가게 하고 싶다" → 가맹 문의. 소비자 경험의 품질이 곧 가맹 모집의 핵심.

## 5. 의사결정에 연결되는 체크리스트

4강 핵심: "페르소나는 벽에 붙여놓고 안 보면 쓸모없다. 의사결정에 연결되어야 한다."

| 의사결정 | 타깃이 결정하는 것 |
|---------|----------------|
| 메뉴 구성 | Job 1(빠른 점심)에 맞게 **핵심 메뉴 집중**, 주문→서빙 10분 이내 |
| 가격대 | ₩11,000 유지. 리틀하노이(₩3,900)와 다른 세그먼트 |
| 매장 위치 | 오피스 상권, 직장인 점심 동선 |
| 영업시간 | 점심+저녁. 혼밥 저녁도 커버 |
| LP 설계 | 소자본 Primary → 비용 투명 공개를 히어로 근처 배치 |
| SNS 콘텐츠 | Job 2(혼밥 경험) → 카운터에서 한 그릇 먹는 장면, 조리 과정 |
| 인테리어 | Job 1+2 → 카운터에서 조리 과정 보이는 구조 유지·강화 |

## 6. 분기마다 점검할 질문 (4강)

1. 최고의 고객 5명은 누구였고, 공통점은?
2. 가장 많이 받은 칭찬과 불만은?
3. 혼밥:동행 비율은? (카운터:테이블 사용률)
4. 점심:저녁 매출 비율은?
5. 가맹 문의 중 소자본:탈직장인 비율은?
