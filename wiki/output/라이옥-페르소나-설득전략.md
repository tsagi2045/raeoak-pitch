# 라이옥 최적 페르소나 및 설득 전략

> **주의**: 미팅 전 방향성 가이드. 세부 기획에 따라 달라질 수 있음.

## 페르소나 우선순위 변경

[[raw-03-summary]] 원래 순서(A > B > C)를 [[라이옥]] 특성에 맞게 재조정:

| 순위 | 일반 분석 | 라이옥 특화 | 이유 |
|------|----------|-----------|------|
| **Primary** | 탈직장인 | **탈직장인 (유지)** | 투자 규모 매칭 + 정통성에 공감 |
| **Secondary** | 다점포 운영자 | **소자본 첫 창업 (↑)** | "밀착 지원"이 최대 메리트 + 수도권 선점 |
| **3순위** | 소자본 | **다점포 운영자 (↓)** | ROI 데이터 부족, 비점주 운영 어려움 |

## 적합도 매트릭스

| 평가 기준 | 탈직장인 | 다점포 | 소자본 |
|----------|:---:|:---:|:---:|
| "정통성" USP에 공감 | ★★★★★ | ★★☆ | ★★★☆ |
| "작지만 밀착" 수용도 | ★★★★☆ | ★★☆ | ★★★★★ |
| 투자 규모 매칭 | ★★★★★ | ★★★★☆ | ★★☆ |
| 수도권 초기 가맹 매력 | ★★★★☆ | ★★★☆ | ★★★★★ |
| 데이터 부족 감내 | ★★★☆ | ★☆ | ★★★★☆ |

**다점포 하락 이유**: 의사결정 1순위 "수익성 데이터"(35%)인데 라이옥은 데이터 부족. "비점주 상주"를 원하나 "13시간 육수 + 원칙 고수"는 점주 참여 필요.

## Primary: 탈직장인 "가치 지향형"

### 재정의

| 일반 탈직장인    | 라이옥형 탈직장인             |
| ---------- | --------------------- |
| "안 망하면 된다" | "할 거면 제대로 하고 싶다"      |
| 수익 안정성 최우선 | 수익 + **자부심** 둘 다      |
| 대형 브랜드 선호  | 대형 브랜드 공장식 맛에 **회의적** |
| 숫자로 설득     | 숫자 + **스토리**로 설득      |

### LP 설득 흐름

1. **공감** — "퇴직 후 프랜차이즈를 알아보셨나요? 다 비슷해 보이지 않던가요?"
2. **차별화** — "베트남인이 80% 찾아오는 쌀국수집이 있습니다"
3. **원칙** — "재료가 원칙에 어긋나면 그 날 판매를 중단합니다"
4. **증거** — 다이닝코드 4.8, 부산 10년, 이홍렬TV
5. **안심** — "10개 매장이기에 가능한 1:1 밀착 지원, 대표가 직접"
6. **기회** — "수도권 초기 가맹, 지금이 합류 시점"
7. **CTA** — "수익 시뮬레이션 받기"

### 핵심 메시지

> "대형 프랜차이즈에서는 불가능한 것을 합니다. 13시간 육수, 매일 5종류, 재료 없으면 판매 중단. 이 원칙이 10년간 부산에서 줄을 만들었습니다."

### 전환 장벽 해소

| 장벽 | 대응 |
|------|------|
| "10개 매장이라 불안" | "[[미스사이공]]은 274→90. 많은 게 답이 아닙니다" |
| "부산 브랜드, 서울에서 될까" | "미아점 오픈 + 베트남인이 인정한 맛은 지역 무관" |
| "배우자 설득" | "다이닝코드 맛 4.8" + "함께 본점 방문" |

## Secondary: 소자본 "동반 성장형"

### 재정의

| 일반 소자본 | 라이옥형 소자본 |
|-----------|---------------|
| "저렴하면 좋겠다" | "저렴하면서 **제대로 된** 곳" |
| 초기 비용 최우선 | 초기 비용 + **본사가 진짜 도와주는지** |
| 유튜브/인스타 정보 수집 | 브랜드 감성과 스토리에 반응 |

### LP 설득 흐름

1. **공감** — "첫 창업, 뭘 해야 할지 막막하시죠?"
2. **안심** — "라이옥은 10개 매장이기에 한 분에게 집중합니다"
3. **감성** — "라이(Lai)와 옥자, 두 사람의 이름으로 시작된 쌀국수"
4. **트렌드** — "건강식 + 1인 가구 + 배달 = 지금 시작" ([[시장-기회]])
5. **실체** — 13시간 육수, 베트남인 80%, 줄 서는 사진
6. **기회** — "수도권 초기 가맹, 선점 기회"
7. **CTA** — "무료 창업 상담 신청"

### 핵심 메시지

> "처음이라서 괜찮습니다. 10개 매장의 본사는 100개 매장의 본사보다 당신에게 더 많은 시간을 씁니다."

### 전환 장벽 해소

| 장벽 | 대응 |
|------|------|
| "7천만원으로 가능한가" | 소형 모델 유무 미팅 확인 (B3). 가능하면 킬러 포인트 |
| "사업 경험 없다" | "교육장 + SV 밀착 사후관리" |
| "부모님 설득" | "다이닝코드 4.8 + 10년 부산 맛집" = 부모님 세대도 이해 |

## 3순위: 다점포 운영자 (축소 대응)

LP 별도 섹션 없음. 미팅 B2(매출 데이터 공개 범위) 결과에 따라:
- 공개 가능 → "기존 외식업 경험자를 위한 실적 자료" 섹션 추가
- 비공개 → 이 페르소나 사실상 포기 (데이터 없이 ROI 타겟 설득 불가)

## LP CTA 분기 설계

```
[히어로] — "베트남인이 줄 서는 쌀국수"
    ↓
[브랜드 스토리] — 라이+옥자, 13시간, 80% 베트남인
    ↓
[시장 기회] — 블루오션 데이터 (공통)
    ↓
[본사 지원] — "10개 매장의 밀착 지원" (탈직장인 + 소자본)
    ↓
[투자 비용] — 공개 가능 범위 (미팅 B1 확인)
    ↓
[Social Proof] — 다이닝코드 4.8, 줄 사진, 미디어
    ↓
[CTA 분기]
  ○ "수익 시뮬레이션 받기"  ← 탈직장인
  ○ "무료 창업 상담 신청"   ← 소자본
```
