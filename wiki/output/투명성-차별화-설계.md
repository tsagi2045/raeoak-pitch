# 포메인 벤치마킹 + 투명성 차별화 — 섹션별 데이터 배치 설계

> **주의**: 미팅 전 방향성 가이드. 클라이언트 미팅 결과와 세부 기획에 따라 구체적 내용은 달라질 수 있음.

## 핵심 논리

[[포메인]]은 업계 유일의 독립 LP(phomeinfc.com)를 가졌지만 3가지 약점이 있다:

| 포메인의 약점 | 우리의 투명성 공격 |
|-------------|-----------------|
| 비용 **전면 비공개** | 투자비 범위 + 비용 구성을 LP에 공개 |
| 수익 데이터는 **직영점 사례만** | **정보공개서 기반** 가맹점 평균 매출 공개 |
| AI 입지분석을 **언급만** 하고 LP에서 체험 불가 | 상권 분석 **미리보기** 인터랙티브 기능 |

## 섹션별 설계

### 1. 히어로 (Hero)

- 헤드라인: "치킨 3만개, 커피 4만개 시대 — 쌀국수는 아직 _____개" ([[시장-기회]] 블루오션)
- 서브 카피: 클라이언트 실제 가맹점 수 삽입 (미팅 후 확정)
- 포메인 대비: 포메인은 "19년 전통" 감성. 우리는 **숫자로 시작**

### 2. 시장 기회 (Why Now)

| 데이터 | 수치 |
|--------|------|
| 에스닉 푸드 성장 | 9년간 **5배** |
| 건강식 | 글루텐프리, **300~400kcal** |
| 1인 가구 | **815만** 가구, 혼밥 긍정 **83%** |
| 배달 시장 | **40조원** |
| 쌀국수 점유율 | 외식 프랜차이즈의 **0.35%** |

포메인은 시장 데이터를 LP에 노출하지 않음. 우리는 **공공 통계 기반**으로 증명.

### 3. 수익 모델 ← 핵심 투명성 섹션

| 데이터 | 법적 주의 |
|--------|----------|
| 가맹점 평균 매출 (정보공개서 기준) | "정보공개서 기준" 필수 명시 |
| 영업이익률 추정 10~15% | "추정"임 명시, 보장 표현 금지 |
| 투자금 회수 기간 | 업계 평균 31.4개월 vs 클라이언트 실제 |
| 프랜차이즈 vs 비프랜차이즈 | 3.23억 vs 2.3억 ([[시장-기회]]) |

포메인: 직영점 매출만. 우리: **가맹점** 평균 매출 + 정보공개서 기반 검증.

### 4. 투자 비용 ← 두 번째 핵심 투명성 섹션

- 총 창업비용 범위: "8,000만~1.2억원 (매장 규모에 따라)" — 포메인은 완전 비공개
- 비용 구성 분해: 가맹비 / 교육비 / 보증금 / 인테리어 각각 표시
- 업계 비교: 경쟁사 평균 대비 위치 (막대그래프)
- 인테리어 비중: "전체의 85% → 소형 매장으로 절감 가능" ([[가맹-조건]])

7개 경쟁사 중 가맹비를 공개한 곳은 [[미분당]] 1곳뿐. 비용 구성 분해만으로 업계 최고 투명성.

### 5. 운영 시스템 + 교육/지원

- 교육: 기간, 커리큘럼, 수료율 (포메인은 "교육장 운영" 언급만)
- SV: 방문 주기, 지원 범위 구체화
- 물류: 공급망 도식 + 배송 주기
- AI 상권 분석: LP에서 주소 입력 → 간이 분석 결과 즉시 (포메인은 "도입 예정"만)

### 6. 가맹점 성과 (Social Proof)

- 점주 인터뷰 영상 3개: 페르소나 A/B/C에 대응 ([[raw-03-summary]])
- 가맹점 실제 매출 범위 (포메인: 직영점만)
- 폐점률 — 낮으면 공개, 높으면 비공개

### 7. CTA (전환)

[[랜딩페이지-전략]] 페르소나별 CTA를 선택형 폼으로 통합:
- "수익 시뮬레이션 받기" ← Primary
- "가맹점 실적 자료 요청" ← 다점포 운영자
- "무료 창업 상담 신청" ← 소자본

## 전략의 열쇠: 미팅 질문 D5

> 클라이언트가 "투자비와 매출을 LP에 공개해도 된다"고 승인해야 투명성 전략이 작동한다.

| 섹션 | 필요 데이터 | 미팅 질문 |
|------|-----------|----------|
| 수익 모델 | 가맹점 평균 매출, 영업이익률 | A4 |
| 수익 모델 | 폐점률 | A5 |
| 투자 비용 | 항목별 비용 | B1 |
| 투자 비용 | **공개 가능 범위** | **D5** |
| 가맹점 성과 | 점주 인터뷰 | D3 |
| 상권 분석 | 분석 도구/데이터 보유 여부 | B6 |
