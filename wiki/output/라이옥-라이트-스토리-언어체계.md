# 라이옥 라이트 브랜드 스토리 + 언어 체계

브랜딩 커리큘럼 5강(StoryBrand) + 6강(언어 체계) 적용.

---

## 1. 내러티브 (상위 세계관)

내러티브 = 모든 개별 스토리를 관통하는 **브랜드의 믿음**.

### 1.1 라이옥 전체 내러티브 (스탠다드+라이트 공통)

> **"진짜 맛에는 시간과 전통이 필요하다."**

이 믿음 아래 모든 콘텐츠가 정렬:
- 15시간 육수 → 시간의 가치
- 하노이 50년 전수 → 전통의 가치
- 베트남인 80% 고객 → 정통의 증거
- 재료 부족 시 판매 중단 → 원칙의 가치

### 1.2 라이트 전용 내러티브 (하위)

> **"진짜는 빠를 수도 있다."**

스탠다드와 공유하는 "진짜"에 라이트만의 "빠르게"를 추가. 모순처럼 들리지만 이것이 정확히 라이트의 존재 이유:
- 15시간 걸리는 육수를 → 본사가 만들고
- 고객은 10분 안에 → 카운터에서 경험한다

"빠르다"가 "대충"이 아니라 **"시스템의 결과"**라는 것이 메시지의 핵심.

---

## 2. StoryBrand 7단계 — 소비자 (B2C)

### 영웅 = 고객

바쁜 수도권 직장인. 점심 40분. 매일 "뭐 먹지"가 고민. 기준은 있지만 시간이 없다.

### 문제 (3겹)

| 겹 | 문제 |
|----|------|
| **외적** | 빠르면서 맛있는 점심을 찾기 어렵다 |
| **내적** | "프랜차이즈는 다 비슷하고, 맛집은 줄이 길고, 편의점은 허전하다" — 선택의 피로 |
| **철학적** | 바쁘다고 대충 먹어야 하는 건 불합리하다 |

고객이 지갑을 여는 건 **내적 문제** 때문. "오늘도 대충이겠지"라는 피로감을 깨주는 것이 핵심.

### 가이드 = 라이옥 라이트

| 자질 | 표현 |
|------|------|
| **공감** | "바쁜 하루, 한 끼만이라도 제대로 드시길 바랍니다" |
| **권위** | "하노이 50년 전통, 15시간 육수, 부산 10년 검증" |

공감 없이 권위만 → 거만. 권위 없이 공감만 → 무능. 둘 다 있어야.

### 계획

고객이 해야 할 일이 명확해야 행동 장벽이 낮아진다:

1. **카운터에 앉는다** (혼밥도 편한 구조)
2. **키오스크로 주문** (빠르고 간편)
3. **눈앞에서 조리되는 과정을 본다** (기다림이 경험이 되는 순간)
4. **첫 한 입** (15시간 육수의 진실의 순간)
5. **소스·고수로 취향 커스텀** (나만의 한 그릇)

### 행동 촉구

모호하면 안 됨:
- "첫 방문이라면 소고기 쌀국수부터. 고수는 요청하시면 듬뿍."
- "점심 세트는 주문 후 7분 안에 나옵니다."

### 성공

- "국물 한 방울까지 비웠어요" (완식 장면)
- "여기 육수 진짜 깊더라" (지인에게 추천)
- "매주 화요일은 라이옥" (루틴이 된 고객)

### 실패

행동하지 않으면:
- "오늘도 편의점 도시락으로 때웠다"
- "맵고 기름진 점심에 속이 더부룩하다"

## 3. StoryBrand 7단계 — 가맹 (B2B, 소자본 Primary)

### 영웅 = 예비 창업자

25~35세, 가용 자산 3,000~7,000만원. 직장에 한계를 느끼고 "내 가게"를 꿈꾸지만 실패가 두렵다.

### 문제 (3겹)

| 겹 | 문제 |
|----|------|
| **외적** | 적은 돈으로 할 수 있는 검증된 창업 아이템을 못 찾겠다 |
| **내적** | "나같은 초보가 식당을 할 수 있을까" — 자격지심 |
| **철학적** | 처음이라고 꿈을 포기해야 하는 건 불공평하다 |

### 가이드 = 라이옥

| 자질 | 표현 |
|------|------|
| **공감** | "처음이라서 괜찮습니다. 우리도 부산 골목에서 시작했습니다" |
| **권위** | "10년 검증, 13개 매장, 본사 육수 중앙공급, 삼성 웰스토리 물류" |

### 계획

1. **무료 상담 신청** (부담 없이 시작)
2. **라이트 모델 설명 + 비용 투명 공개** (불확실성 해소)
3. **본점 방문 + 맛 체험** (백문이불여일견)
4. **상권 분석 → 계약 → 교육** (체계적 단계)
5. **오픈 + SV 밀착 지원** (혼자가 아닌)

### 행동 촉구

- "무료 창업 상담 신청"
- "라이트 모델 상세 자료 받기"
- "5,000만원 이하 창업, 가능합니다 — 상담받기"

### 성공

- "내 가게 앞에 줄이 서는 날" (비전)
- "6개월 만에 손익분기점을 넘었습니다" (실체)
- "부모님이 자랑스러워하시더라" (감정)

### 실패

- "또 한 번의 면접, 또 한 번의 불합격"
- "남들은 자기 사업 하는데 나만 제자리"

## 4. 슬로건 후보 (6강)

### 조건

- 포지셔닝("정통+빠른+밀착")의 번역
- 7음절 이내
- 라이옥 없이도 우리 것인 줄 알 수 있는

### 소비자용 슬로건 후보

| # | 슬로건 | 음절 | 포인트 |
|---|--------|------|--------|
| S1 | **"15시간의 깊이, 눈앞에서"** | 10 | 시간+밀착 |
| S2 | **"진짜는 빠를 수도 있다"** | 9 | 내러티브 직역 |
| S3 | **"한 그릇의 하노이"** | 7 | 정통성+간결 |
| S4 | **"가까이서, 깊이를"** | 6 | 밀착+깊이 |
| S5 | **"바쁜 하루, 진짜 한 그릇"** | 9 | 타깃 상황+정통성 |

### 가맹용 슬로건 후보

| # | 슬로건 | 포인트 |
|---|--------|--------|
| F1 | **"타협 없는 맛, 부담 없는 시작"** | 정통성+소자본 |
| F2 | **"5천만원으로 시작하는 10년의 맛"** | 구체 숫자+검증 |
| F3 | **"처음이라서 괜찮습니다"** | 기존 메시지 계승, 공감 |
| F4 | **"작지만 진짜인 내 가게"** | 기존 포지셔닝 계승 |

### 권고

- 소비자: **S4 "가까이서, 깊이를"** — 가장 짧고 함축적. "가까이서"=카운터 밀착, "깊이를"=15시간 육수의 깊이. 광고에서 확장 가능 ("15시간의 깊이를, 눈앞에서").
- 가맹: **F1 "타협 없는 맛, 부담 없는 시작"** — 소비자 경험(맛 타협 없음)과 창업 장벽(비용 부담 없음)을 동시에 담음.

## 5. 톤 앤 보이스 (6강)

### Voice (일관된 성격)

> **자신감 있되 겸손. 깊되 간결. 정통적이되 접근적.**

키워드 3개: **깊이 · 절제 · 자부심**

- "깊이": 15시간이라는 시간이 담긴 자신감
- "절제": 군더더기 없음. 말이 적지만 의미가 있음
- "자부심": 떠들지 않아도 맛이 증명하는 자세

### Tone (상황별 조절)

| 상황 | 톤 | 예시 |
|------|-----|------|
| 매장 내 사이니지 | 절제 + 자부심 | "15시간." (한 마디로 끝) |
| 인스타그램 | 깊이 + 감성 | "오늘도 새벽, 육수가 시작됩니다." |
| 배달앱 메뉴 설명 | 명확 + 전문적 | "하노이 정통 방식. 소뼈·양지·향신료 15시간 추출 육수." |
| 네이버 플레이스 | 간결 + 정보 | "카운터에서 즐기는 정통 쌀국수. 주문 후 7분 서빙." |
| LP (소비자) | 자부심 + 초대 | "한 그릇의 하노이를 눈앞에서 경험하세요." |
| LP (가맹) | 공감 + 신뢰 | "처음이라서 괜찮습니다. 10년의 맛이 함께합니다." |
| 고객 응대 | 따뜻함 + 자연스러움 | "맛있게 드셨어요? 다음에는 고수 추가도 추천드려요." |

### 금지어 / 필수어

| 금지어 ❌ | 대신 쓸 말 ✅ | 이유 |
|----------|------------|------|
| "저렴한" | "합리적인" / "부담 없는" | 정통성 브랜드에 "저렴"은 가치 훼손 |
| "퓨전" | "정통" / "현지식" | 라이옥의 핵심이 정통성 |
| "간편한 한 끼" | "진짜 한 그릇" | "간편"은 대충의 뉘앙스 |
| "인스턴트" | (사용 금지) | 15시간 육수의 대척점 |
| "맵기 조절" | "취향 커스텀" | 브랜드 톤 유지 |
| "세트 메뉴" | "한 그릇 코스" / "라이옥 세트" | 브랜드명 연결 |
| "소자본" (LP에서) | "합리적 투자" / "부담 없는 시작" | 가맹점주에게 "소자본"은 스스로를 축소하는 느낌 |
| "1인 매장" | "내 가게" / "나만의 매장" | 소유감·자부심 부여 |

| 필수어 ✅ | 사용 맥락 |
|----------|---------|
| "15시간" | 모든 매체에서 반복. 핵심 기억 장치 |
| "정통" | 한국식이 아님을 명시할 때 |
| "눈앞에서" | 카운터 밀착 경험 표현 |
| "깊이" | 육수의 맛, 시간의 가치 |
| "하노이" | 지역 정통성 연결 |
| "검증" | 10년, 베트남인 80% 등 증거 |

## 6. 개별 스토리 후보 (콘텐츠용)

내러티브("진짜는 빠를 수도 있다") 아래 개별 스토리:

| # | 스토리 | 매체 | 내러티브 연결 |
|---|--------|------|-------------|
| 1 | "새벽 4시, 공장에서 육수가 시작됩니다" | 인스타 릴스 | 시간의 가치 |
| 2 | "왜 우리는 15시간을 줄이지 않기로 했는가" | LP 브랜드 섹션 | 원칙의 가치 |
| 3 | "라이(Lai)와 옥자 — 두 사람의 이름으로 시작된 쌀국수" | LP 스토리 | 전통의 가치 |
| 4 | "미아점 첫날, 첫 손님의 한 마디" | SNS/LP | 검증의 순간 |
| 5 | "카운터에서 3분, 눈으로 먹는 시간" | 인스타 릴스 | 밀착 경험 |
| 6 | "부산에서 서울까지, 육수는 같습니다" | LP/배달앱 | 품질 일관성 |
| 7 | "처음 창업한 김○○ 점주의 6개월" | LP 가맹 | 성공 스토리 |

## 7. 요약: 언어 체계 1장 가이드

| 요소 | 내용 |
|------|------|
| **내러티브** | "진짜는 빠를 수도 있다" |
| **소비자 슬로건** | "가까이서, 깊이를" |
| **가맹 슬로건** | "타협 없는 맛, 부담 없는 시작" |
| **Voice** | 깊이 · 절제 · 자부심 |
| **필수어** | 15시간, 정통, 눈앞에서, 깊이, 하노이, 검증 |
| **금지어** | 저렴, 퓨전, 간편, 인스턴트, 소자본(LP) |

이 한 장이 있으면 메뉴판, SNS, 배달앱, LP, 매장 내 사이니지까지 **누가 써도 라이옥 라이트답게** 된다.
