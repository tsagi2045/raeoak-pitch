# 호앙비엣 (Hoang Viet)

베트남 정통 쌀국수 프랜차이즈. 불경기 속에서도 급성장하고 있는 후발 브랜드.

## 핵심 정보

| 항목 | 내용 |
|------|------|
| 매장 수 | 약 20개 (수도권 중심) |
| 성장 속도 | 최근 6개월간 11개 신규 오픈 |
| 포지셔닝 | "진정한 베트남 쌀국수" — 원본성(Originality) 강조 |

## 차별화 포인트

- **베트남 현지 공장에서 직접 제조**한 쌀국수 면과 전용 소스
- 본사 **직수입 식재료 + 자체 물류** 시스템
- 원가 경쟁력과 맛 차별화를 동시에 확보

## 매장 형태 (2026-04-13 확인)

| 항목 | 내용 |
|------|------|
| 매장 규모 | **2층 구성, 넓은 매장** (예: 동국대점 서울 중구 서애로1길 14) |
| 1층 좌석 | 바 테이블(기다란 좌석, 혼밥용) + 일반 테이블. 스테인리스 테이블 |
| 2층 좌석 | 테이블 2개 연결, 4인+ 그룹 대응 |
| 인테리어 | 베트남 현지 골목 식당 감성. 노란 간판, 전통 갓 장식, 패턴 타일 |
| 주문 방식 | 종이 체크식 |
| 가격대 | 쌀국수 ₩10,500~₩12,000, 반쎄오 ₩3,500/개 |
| 리뷰 키워드 | 현지감성, 가성비, 혼밥 최적화, 깊은 국물 맛 |

→ **대형 매장 모델**. 소형 카운터 전문이 아님. 라이옥 라이트와 매장 형태 차별화 가능.

## 가맹점 지원

- 안정적인 공급망 제공
- 체계적인 운영 매뉴얼
- 지속적인 매출 관리 지원

## [[라이옥]]과의 비교

| 항목 | 호앙비엣 | 라이옥 |
|------|---------|--------|
| 매장 수 | ~20개 | ~10개 |
| 지역 | 수도권 중심 | 부산 기반, 수도권 진출 시작 |
| 차별화 | 현지 공장 직제조·자체 물류 | 13시간 육수, 베트남인 80% 고객 |
| 확장 속도 | 6개월간 11개 오픈 | 안정적 확장 추구 |

유사한 규모·포지셔닝의 경쟁사로, 호앙비엣의 급속 확장 전략 vs 라이옥의 정통성 기반 전략이 대비됨.
