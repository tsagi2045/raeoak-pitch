# 포메인 (Phomein)

한국 쌀국수 프랜차이즈의 원조 브랜드(2006~). 업계 유일의 가맹 전용 독립 사이트 운영.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 설립 | 2006년 (19년 역사) |
| 운영사 | (주)데일리킹 |
| 가맹점 수 | ~100개 (2025년 기준) |
| 총 창업비용 | 비공개 ("합리적 비용" 강조) |
| 포지셔닝 | 프리미엄 정통 베트남 쌀국수 |
| 브랜드 사이트 | phomein.com |
| 가맹 전용 사이트 | **phomeinfc.com** (유일) |

## 핵심 성과

- 누적 고객 수 **1억명** 돌파
- 2024년 연간 고객 수 500만명
- 가맹점 연매출: 2022년 592억 → 2023년 643억 → 2024년 596억
- 직영점 월매출 사례: 에버랜드점 3.8억, 정자본점 1.8억
- 2025 매일경제 100대 프랜차이즈 선정

| 전략 | 상세 |

| **교육장** | 분당 정자동 전문 교육장 운영 |

## 우리에게 주는 시사점

### 벤치마킹 포인트
- **독립 가맹 사이트**: 우리도 독립 도메인/서브도메인으로 LP 구축 가능
- **AI 입지 분석**: "상권 분석 미리보기" 기능을 LP에 탑재하면 차별화
- **RED 모델**: 소형 매장 옵션이 소자본 타겟에 효과적

### 차별화 가능 포인트
- 비용 비공개 → 우리는 투자비 범위 공개로 신뢰 차별화
- SPA 방식으로 SEO 약점 → 우리는 SSR/정적 생성으로 검색 유리
- 수익 데이터 부족 → 정보공개서 기반 검증 데이터 제시
