# 미분당 (Mifendang)

가맹점 수 기준 한국 쌀국수 프랜차이즈 1위. 한국인 입맛에 맞춘 '한국식 쌀국수' 포지셔닝.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 설립 | 2018년 |
| 가맹점 수 | ~171개 (업계 1위) |
| 포지셔닝 | 한국식 쌀국수 (프리미엄) |
| 총 창업비용 | 약 1억 92만원 |
| 가맹비 | 1,000만원 |
| 교육비 | 100만원 |
| 보증금(물류) | 500만원 |
| 최소 점포 | 약 13평 |
| 웹사이트 | mifendang.com (SSL 미적용) |

## 특징

- 2018년 설립으로 비교적 짧은 역사에도 **가맹점 수 1위** 달성
- "한국인 입맛에 맞게 재해석한 쌀국수"로 차별화
- 마이프차 플랫폼에 비교적 상세한 비용 정보 공개
- 양도양수 매물 활발 (월매출 4,500만원, 월수익 1,400~1,800만원 사례)

## 가맹 모집 전략

- 홈페이지(mifendang.com) 내 가맹 안내 섹션 — 별도 랜딩페이지 없음
- 마이프차 등 프랜차이즈 정보 플랫폼 활용
- 독립 가맹 전용 랜딩페이지는 미보유

## 우리에게 주는 시사점

- **비용 공개의 효과**: 업계에서 비교적 투명하게 비용을 공개 → 정보 플랫폼에서 노출 유리
- **벤치마크**: SOM 낙관적 시나리오(80개+ 가맹점)의 참고 대상
- **약점**: SSL 미적용, 독립 랜딩페이지 부재 → 차별화 가능 지점
