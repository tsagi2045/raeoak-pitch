# 에머이 (Emoi)

"가장 가까운 베트남"을 슬로건으로 서울 중심으로 확장하는 쌀국수 프랜차이즈.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 설립 | 2017년 |
| 가맹점 수 | ~37개 |
| 총 창업비용 | 약 1억 2,403만원 |
| 매출 (2024) | 63억원 |
| 포지셔닝 | "가장 가까운 베트남" — 현지 감성 |
| 웹사이트 | emoikorea.com (imweb 템플릿) |

## 특징

- 서울 중심 확장 전략 — 수도권 밀집
- 2017년 설립, 꾸준한 성장세
- imweb 기반 웹사이트 (템플릿 한계)
- 베트남 현지 감성을 강조한 인테리어/메뉴

## 가맹 모집 전략

- 홈페이지 내 가맹 안내 섹션
- imweb 템플릿 기반 → 디자인 커스터마이징 한계
- 독립 랜딩페이지 미보유

## 우리에게 주는 시사점

- **SOM 기본 시나리오 벤치마크**: 20~80개 가맹점 단계의 참고 대상
- **imweb 한계**: 템플릿 기반 사이트의 차별화 어려움 → 맞춤 LP로 차별화 가능
- **서울 중심 전략**: 지역 집중 전략의 참고 사례 (클라이언트 모집 전략에 따라)
