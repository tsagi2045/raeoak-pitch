# 미스사이공 (Miss Saigon)

급속 확장 후 대량 폐점을 경험한 쌀국수 프랜차이즈. 업계에 "천천히 확실하게"라는 교훈을 남긴 사례.

## 기본 정보

| 항목              | 내용                  |
| --------------- | ------------------- |
| 설립              | 2016년               |
| 가맹점 수 (피크)      | 274개                |
| 가맹점 수 (2024.11) | ~90개                |
| 총 창업비용          | ~9,600만원 (업계 평균 수준) |
| 포지셔닝            | 저비용 1인 운영 특화        |
| 웹사이트            | miss420.net (보안 이슈) |

## 흥망 타임라인

- **2016**: 설립. 노량진 노점에서 시작한 창업자 스토리
- **~2022**: 빠른 가맹 확장 → **274개 매장** 달성
- **2023~2024**: 대량 폐점 → **90개로 급감** (67% 감소)
- 결혼이민자의 성공 스토리가 브랜드 정체성이었으나, 급속 확장의 부작용으로 품질 관리 실패

## 교훈

| 문제 | 시사점 |
|------|--------|
| **급속 확장** | 양적 성장보다 질적 성장 우선. 가맹점 관리 역량 초과 시 브랜드 전체 훼손 |
| **품질 편차** | 본사 물류/교육/SV 체계가 확장 속도를 따라가지 못하면 폐점률 급증 |
| **업계 신뢰 저하** | 한 브랜드의 실패가 쌀국수 카테고리 전체에 대한 불신으로 확산 가능 |

## 가맹 모집 전략

- 기본적인 웹사이트 운영 (보안 이슈 존재)
- 1인 운영 가능 모델 강조
- 창업자 스토리텔링 활용 (감정적 연결)

## 우리에게 주는 시사점

- **반면교사**: 랜딩페이지에서 "안정적 성장"을 강조하는 근거로 활용
- **데이터 투명성**: 폐점률을 숨기지 않고 낮은 폐점률을 오히려 어필
- **클라이언트 미팅**: 클라이언트의 확장 전략이 급진적이지 않은지 확인
- **차별화**: "우리는 다릅니다"의 근거로 SV 체계, 가맹점 관리 시스템 강조
