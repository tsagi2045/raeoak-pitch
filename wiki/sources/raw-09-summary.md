# 소스 요약: 라이옥 공식 가맹 브로슈어

라이옥 가맹본부가 예비 창업주 대상으로 제작한 공식 가맹 브로슈어(14p). 기존 웹 조사([[raw-07-summary]])에서 확인 불가했던 **가맹 비용, 수익률, CI, 팀 구조** 등 내부 데이터를 포함.

## 핵심 Takeaway

1. **2개 타입 가맹 모델** — 라이트(10~15평, 1인 소형) vs 스탠다드(25평+, 풀 운영). 소자본 창업 트렌드 대응.
2. **영업이익률 32~34%** — 배달 제외 기준. 라이트(34%)가 스탠다드(32%)보다 높음. 인건비 비율 차이(10% vs 15%).
3. **원재료비 38% 고정** — 모든 매출 구간에서 동일. 본사 식자재 공급 가격 표준화.
4. **4팀 조직 구조** — 가맹본부(총괄) + 교육·개발 + 슈퍼바이징 + 디자인·마케팅.
5. **삼성 웰스토리 물류** — 전국 네트워크 활용. 전국 확장의 물류 기반.

### 가맹 절차 (8단계)

가맹상담 → 점포·상권분석 → 가맹계약 → 인테리어 도면 → 인테리어 시공 → 장비·물품 입고 → 현장 교육 → 매장 오픈
