# 소스 요약: 브랜딩 딥다이브 커리큘럼

10강 구성의 브랜딩 이론 커리큘럼. 모든 예시가 라이옥 쌀국수 프랜차이즈에 맞춰 구성됨. 리브랜딩 프로젝트의 이론적 기반이자 단계별 체크리스트로 활용.

## 핵심 Takeaway

1. **브랜딩 = 인식의 게임** — 로고가 아니라 "고객이 느끼는 것"이 브랜드. 식별(로고) → 약속(기대치) → 의미(자기표현) 3겹 구조.
2. **포지셔닝이 모든 것의 상위 의사결정** — 멘탈 슬롯(고객 머릿속 2~3개 브랜드)에서 빈 칸을 찾아 대표가 되는 것. "우리가 뭘 잘하느냐"가 아닌 "고객이 한 단어로 뭐라 부르느냐".
3. **비주얼은 마지막** — 포지셔닝 → 타깃 → 스토리 → 언어 → 비주얼 순서. 로고부터 만드는 것은 "순서 실수".
4. **브랜드 경험이 곧 브랜드** — 모든 접점의 총합. 피크-엔드 법칙으로 핵심 순간과 마지막 인상 설계.
5. **리브랜딩은 진화** — 리프레시(옷만) < 리포지셔닝(카테고리 재정의) < 오버홀(전면 재구축). "바꾸지 않을 것"을 먼저 정하는 게 핵심.

## 10강 프레임워크 요약

### Phase 1: 기반 설계 (1~4강)

| 강 | 핵심 질문 | 프레임워크 |
|---|---------|-----------|
| 1강 | 우리 브랜드는 뭔가? | 식별 → 약속 → 의미 3겹 |
| 2강 | 고객 머릿속 어디에 앉을 것인가? | 멘탈 슬롯, 포지셔닝 공식, 카테고리 창조 |
| 3강 | 의도와 현실의 간극은? | 캐퍼러 프리즘 6면 (체격/개성/문화/관계/반영/자아상) |
| 4강 | 누구의 어떤 문제를 해결하는가? | 3층 타깃 (인구통계→사이코그래픽→JTBD), 배제의 기술 |

### Phase 2: 표현 체계 (5~7강)

| 강 | 핵심 질문 | 프레임워크 |
|---|---------|-----------|
| 5강 | 어떤 이야기로 전달할 것인가? | StoryBrand 7단계 (영웅=고객, 가이드=브랜드), 내러티브 vs 스토리 |
| 6강 | 어떤 말투로? | 네이밍 5유형, 슬로건 3조건, 톤앤보이스, 금지어/필수어 |
| 7강 | 어떤 모습으로? | 로고/색상/타이포/이미지스타일/공간디자인, 브랜드 가이드라인 최소버전 |

### Phase 3: 경험 설계 (8강)

| 강 | 핵심 질문 | 프레임워크 |
|---|---------|-----------|
| 8강 | 어떤 경험으로 증명할 것인가? | 접점 맵핑, 진실의 순간(ZMOT/FMOT/SMOT), 피크-엔드 법칙, 프랜차이즈 일관성 3방법 |

### Phase 4: 진화 (9~10강)

| 강 | 핵심 질문 | 프레임워크 |
|---|---------|-----------|
| 9강 | 언제 다시 세울 것인가? | 5가지 신호, 3단계 깊이(리프레시/리포지셔닝/오버홀), 4단계 프로세스 |
| 10강 | 작은 브랜드에 어떻게 적용하는가? | 퍼스널 브랜딩, 콘텐츠 3원칙(과정/교육/비교) |

## 라이옥 적용 핵심 (문서 내 제안)

### 포지셔닝 후보 (2강)

- "하노이 50년 정통 육수" → 진짜배기, 정통성
- "15시간 육수의 깊이" → 슬로우푸드 쌀국수
- "베트남 현지인이 직접 조리하는" → 사람 기반 정통성

### 캐퍼러 프리즘 초안 (3강)

| 면 | 라이옥 |
|----|--------|
| 체격 | 베트남풍 인테리어, 오픈키친, 전통 그릇 |
| 개성 | 과묵하지만 실력 있는 장인 |
| 문화 | 하노이 50년 전통, 정통성 |
| 관계 | 현지 맛을 안내하는 가이드 |
| 반영 | 음식 기준이 확실한 사람 |
| 자아상 | "나는 진짜를 먹는 사람이다" |

### 내러티브 후보 (5강)

> "진짜 맛에는 시간과 전통이 필요하다"

### 네이밍 분석 (6강)

"라이옥"은 출처 표시형 이름 — 하노이 원조 매장의 이름(Lai Ngoc)을 그대로 사용. 정통성 포지셔닝과 완벽히 맞물리는 강점. 한국인 발음 보완을 위한 슬로건/부제 필요.

### 리브랜딩 깊이 판단 (9강)

라이옥 현재 상황에 해당하는 신호:
- 신호 1: 타깃 변화 (부산 로컬 → 전국 확장)
- 신호 2: 카테고리 변화 (쌀국수 → 쌀국수+국밥+반미+카페)
- 신호 5: 새로운 성장을 위한 재정립 필요

→ **2단계 리포지셔닝** 적합. 이름·핵심가치 유지, 카테고리 내 가치 재정의.
