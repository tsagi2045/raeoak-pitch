# 소스 요약: 경쟁사 분석

## 원본 개요

쌀국수 프랜차이즈 10개 브랜드의 가맹 조건, 모집 전략, 랜딩페이지를 분석한 벤치마킹 문서. 513줄.

## 핵심 Takeaway

1. **시장은 춘추전국시대**: [[미분당]](171개), [[포메인]](100개), [[미스사이공]](90개) 등 다수 경쟁하나 압도적 1위 부재
2. **독립 랜딩페이지는 블루오션**: 포메인만 가맹 전용 도메인 운영. 나머지는 홈페이지 내 서브 섹션
3. **투명성이 무기**: 대부분 가맹비를 숨기는 시장에서 비용 공개가 차별화 수단
4. **미스사이공의 교훈**: 274개→90개 급락. "천천히 확실하게"가 유리
5. **총 창업비용 범위**: 770만원(헬로베트남) ~ 1억 8,540만원(반포식스)

## 차별화 기회 5가지

1. **가맹 전용 고전환 랜딩페이지** — 포메인 외 부재
2. **비용 투명성** — 투자비 구간 공개로 신뢰 구축
3. **데이터 기반 수익 사례** — 정보공개서 + 점주 인터뷰 조합
4. **디지털 가맹 퍼널 자동화** — 랜딩→관심등록→자동안내→상담예약
5. **AI 상권 분석 미리보기** — 주소 입력 → 즉시 결과
