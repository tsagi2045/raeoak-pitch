# raw-07 요약: 라이옥 프로필

웹 조사 기반 [[라이옥]] 브랜드 프로필 종합. 공식 웹사이트, 언론 기사, 다이닝코드, 네이버 플레이스, 인스타그램 등에서 수집.

## 핵심 Takeaway

1. **정통성 기반 USP**: 고객 80%가 베트남인, 13시간 육수, 매일 5종 이상 육수 별도 제조. 로컬라이제이션 거부하고 현지 맛 고수
2. **부산 거점에서 수도권 진출 시작**: 2015년 본점 이후 부산/경남 중심 10개 매장. 2026년 서울 미아점 오픈으로 수도권 진출 개시
3. **수직 통합 인프라**: 베트남 현지 지사 + 국내 물류창고(부산 강서구) + 기술개발팀 + 온라인 커머스
4. **가맹 웹 현황 미흡**: cafe24 기반 쇼핑몰에 가맹 서브페이지. 독립 LP 없음, 비용/절차 비공개
5. **프랜차이즈 담당 = 우리 클라이언트**: 조민서 (가맹 확장 주도, 미아점 오픈)

## 차별화 포인트 (USP)

- **육수**: 13시간+ 우림, 매일 5종 이상 별도 제조
- **정통성**: 베트남인 고객 80%, 현지인이 인정하는 맛
- **원칙**: 재료 부족 시 당일 판매 중단, 격주 월요일 연구일 운영
- **인프라**: 베트남 현지 지사, 국내 물류창고, 기술개발팀
