# 타겟 오디언스: 쌀국수 프랜차이즈 잠재 가맹점주

## 1. 한국 프랜차이즈 창업자 현황

### 1.1 창업자 연령대 분포

2025년 8월 기준 한국 창업 시장의 연령대별 분포는 다음과 같다 (창업진흥원 통계):

| 연령대 | 창업 건수 | 비율 |
|--------|----------|------|
| 40~49세 | 22,586개 | 25.2% |
| 30~39세 | 22,106개 | 24.6% |
| 50~59세 | 19,392개 | 21.6% |
| 60세 이상 | 13,685개 | 15.3% |
| 30세 미만 | 11,057개 | 12.3% |

**핵심 인사이트**: 30~50대가 전체 창업의 71.4%를 차지하며, 자본력과 경험을 갖춘 중장년층이 창업 시장을 주도하고 있다.

### 1.2 프랜차이즈 창업 동기

소상공인진흥원 프랜차이즈 실태조사 기준:

| 창업 동기 | 전국 | 서울 |
|-----------|------|------|
| 생계 목적 (대안 부재) | 76.1% | 60.3% |
| 생계 + 투자 병행 | 21.1% | 35.3% |
| 투자 목적 | 2.4% | 4.3% |

서울은 투자 목적 비율이 전국 대비 높아, 지역에 따라 창업 동기가 상이하다.

### 1.3 창업 비용 현황

- **외식 프랜차이즈 평균 창업 비용**: 약 9,591만 원 (2025년 중소기업중앙회 조사)
  - 최초 가맹 비용: 2,011만 원
  - 매장 개설 비용: 7,580만 원 (인테리어 4,084만 원 포함)
- **쌀국수 프랜차이즈 (미분당 기준)**: 점포비 제외 약 1억 초반, 점포비 포함 시 1.5~2억 원
- **투자금 회수 기간**: 평균 31.4개월 (약 2년 7개월)
- **월평균 매출**: 약 2,669만 원

### 1.4 예비 창업자 투자 가용 자산

마이프차 2026 트렌드 리포트 (예비 창업자 874명 설문):

| 가용 자산 | 비율 |
|-----------|------|
| 5천만 원 이하 | 29.6% |
| 5천만~1억 원 | 34.7% |
| 1억 원 초과 | 35.7% |

예비 창업자 10명 중 6명 이상이 1억 원 이하의 실속형 창업을 지향하고 있다.

### 1.5 쌀국수/베트남 음식 프랜차이즈 시장 동향

- 에스닉 푸드 시장의 지속적 성장으로 쌀국수 카테고리 수요 확대
- 몬안베띠, 2025 대한민국 소비자 선호도 평가 '창업' 부문 1위 수상
- 한국인 입맛에 맞춘 현지화 메뉴 전략이 성공의 핵심
- 건강식/가심비 트렌드와 맞물려 쌀국수의 소비자 인지도 상승

## 2. 세그먼트 정의

### 2.1 세그먼트 분류 기준

프랜차이즈 잠재 가맹점주를 다음 4가지 기준으로 세그먼트화한다:

| 기준 | 세분화 항목 |
|------|------------|
| **창업 경험** | 첫 창업 vs. 업종 전환 vs. 다점포 확장 |
| **자산 규모** | 소자본(5천만 원 이하) / 중자본(5천만~1.5억) / 고자본(1.5억 이상) |
| **창업 동기** | 생계형 / 투자형 / 라이프스타일형 |
| **연령대** | 20대 / 30대 / 40대 / 50대+ |

### 2.2 세그먼트별 특성 요약

| 세그먼트 | 핵심 특성 | 쌀국수 프랜차이즈 적합도 |
|----------|-----------|------------------------|

## 3. 페르소나

### 3.1 페르소나 A: 김정훈 — "탈직장인 창업자"

#### 배경 스토리

> "15년간 중견 IT 회사에서 마케팅 팀장을 했습니다. 올해 초 구조조정 대상이 되면서 희망퇴직을 선택했어요. 퇴직금 1억 2천만 원과 그간 모아둔 저축을 합치면 2억 원 정도 됩니다. 재취업? 40대 중반에 갈 곳이 마땅치 않더라고요. 아내와 상의 끝에 창업을 결심했는데, 뭘 해야 할지 막막합니다. 일단 실패 확률이 낮은 프랜차이즈를 알아보고 있어요. 쌀국수는 제가 좋아하기도 하고, 건강한 이미지라 앞으로도 수요가 있을 것 같아서 관심이 갑니다."

#### 인구통계

| 항목 | 내용 |
|------|------|
| 이름 | 김정훈 (가명) |
| 나이 | 44세 |
| 성별 | 남성 |
| 거주지 | 경기도 수원시 |
| 가족 | 기혼, 초등학생 자녀 2명 |
| 이전 직업 | 중견 IT기업 마케팅 팀장 (15년 근무) |
| 가용 자산 | 약 2억 원 (퇴직금 1.2억 + 저축 8천만) |
| 월 필요 소득 | 최소 400만 원 (생활비 + 자녀 교육비) |

#### Jobs-to-be-Done

1. **경제적 안정 확보**: 퇴직 후 가족의 생계를 책임질 안정적 수입원을 만들어야 한다
2. **사회적 정체성 회복**: "무직" 상태에서 벗어나 사업주로서의 새로운 정체성을 확립하고 싶다
3. **노후 자산 보전**: 퇴직금을 잃지 않으면서 수익을 창출하는 사업을 운영해야 한다
4. **가족과의 시간 확보**: 직장인 시절보다 유연한 시간 운영이 가능한 일을 원한다

#### Pain Points

| Pain Point | 상세 |
|------------|------|
| 🔴 창업 실패 공포 | "퇴직금이 전 재산인데, 잘못되면 가족 전체가 위기" |
| 🔴 업종 선택 불확실성 | "어떤 업종이 안전한지 판단할 기준이 없다" |
| 🟡 외식업 경험 부재 | "요리도 못하는데 음식 장사를 할 수 있을까" |
| 🟡 입지 선정 불안 | "좋은 자리를 어떻게 찾아야 하는지 모르겠다" |
| 🟡 본사 신뢰 부족 | "가맹 본사들이 수치를 부풀리는 건 아닌지 의심" |
| 🟢 운영 역량 우려 | "직원 관리, 재고 관리 등 처음이라 두렵다" |

#### Gains

| Gain | 상세 |
|------|------|
| 안정적 월 수익 | 월 순이익 500만 원 이상이면 만족 |
| 검증된 시스템 | 본사의 매뉴얼로 외식업 초보도 운영 가능한 체계 |
| 투자금 회수 확신 | 2~3년 내 투자금 회수 가능하다는 근거 |
| 성장 가능성 | 1호점 안정화 후 2호점 확장 가능성 |
| 사회적 인정 | "사장님"이라는 호칭, 자기 사업을 한다는 자부심 |

- 강조점: 검증된 수익 모델, 본사 운영 매뉴얼, 교육 프로그램, 실제 가맹점주 수익 사례
- 톤: 신뢰감, 안정감, 데이터 기반의 객관적 정보 제공
- 회피할 것: 과장된 수익 약속, 감정적 호소

### 3.3 페르소나 C: 이준혁 — "소자본 첫 창업자"

#### 배경 스토리

> "대학 졸업 후 3년간 스타트업에서 일했는데, 솔직히 월급 250만 원으로는 미래가 안 보여요. 부모님이 전세 보증금에서 5천만 원을 지원해주실 수 있다고 하셨고, 제 저축도 2천만 원 정도 있어요. 합쳐서 7천만 원으로 시작할 수 있는 창업을 찾고 있습니다. 배달 위주의 소형 매장을 생각하고 있고, 쌀국수가 요즘 젊은 층에서 인기라서 관심이 갑니다. 근데 사업을 해본 적이 없어서... 본사에서 처음부터 끝까지 잡아줘야 할 것 같아요."

#### Jobs-to-be-Done

1. **경제적 독립**: 월급쟁이로는 불가능한 자산 형성의 기반을 만들고 싶다
2. **커리어 전환**: 남의 회사가 아닌 나의 사업을 가지고 싶다
3. **리스크 최소화**: 적은 자본으로 시작해서 최대한 안전하게 경험을 쌓고 싶다
4. **학습과 성장**: 사업 운영 전반을 배워서 장기적으로 더 큰 사업을 하고 싶다

## 5. 타겟 우선순위

### 5.1 Primary Target: 페르소나 A — 탈직장인 창업자 (김정훈)

**인구통계 프로필**: 35~49세, 5~15년 이상 직장 경력, 퇴직금/목돈 1~2.5억 보유

### 5.2 Secondary Target: 페르소나 B — 다점포 운영자 (박소연)

**인구통계 프로필**: 30~50대, 외식업 운영 경험 3년 이상, 가용 자산 2억 이상

### 5.3 근거

#### Primary로 탈직장인 창업자를 선정한 이유

| 근거 | 상세 |
|------|------|
| **모수가 가장 크다** | 30~50대가 전체 창업의 71.4%를 차지. 희망퇴직/구조조정으로 유입되는 창업자가 지속 증가 추세 |
| **투자 여력이 적합하다** | 쌀국수 프랜차이즈 창업 비용(1~2억)이 이 세그먼트의 자산 범위와 정확히 부합 |
| **전환 가능성이 높다** | 프랜차이즈 창업 동기 중 "생계 + 투자 병행"이 21.1%. 안정적 시스템에 대한 수요가 명확 |
| **LTV가 높다** | 1호점 성공 시 2호점 확장 → 페르소나 B로 전환 가능 |
| **랜딩페이지 전환에 적합** | 정보 탐색 단계에서 온라인 콘텐츠를 적극 소비하는 세그먼트 |

#### Secondary로 다점포 운영자를 선정한 이유

| 근거 | 상세 |
|------|------|
| **높은 계약 전환율** | 이미 외식업에 대한 이해가 있어 의사결정이 빠르다 |
| **낮은 본사 지원 비용** | 기존 운영 경험으로 교육/지원 비용이 절감된다 |
| **다점포 확장 가능성** | 1호점 성공 시 빠른 추가 출점 가능 (설빙 사례: 가맹점의 20%가 다점포) |
| **모수가 제한적** | 전체 잠재 고객 중 비율이 낮아 Primary로는 부적합 |

#### 페르소나 C를 하위 우선순위로 둔 이유

- 자금 규모가 쌀국수 프랜차이즈 최소 투자금에 미달할 가능성이 높음
- 소자본 모델(배달 전문점 등)이 별도로 준비되지 않으면 전환이 어려움
- 다만 소자본 진입 모델이 있다면 볼륨 확보에 유리한 세그먼트이므로, 해당 모델 유무에 따라 우선순위 재조정 필요

## 6. 핵심 시사점

### 6.1 랜딩페이지 설계 시사점

1. **Primary 타겟(탈직장인) 중심으로 설계하되, Secondary(다점포 운영자)를 위한 섹션을 별도로 배치**
   - 메인 스토리라인: "안정적 시스템 → 검증된 수익 → 함께하는 성장"
   - 서브 섹션: "기존 외식업 경험자를 위한 가맹 조건" (다점포 우대)

2. **신뢰 요소가 전환의 핵심**
   - 가맹점주의 76.1%가 "다른 대안이 없어서" 창업한다. 이 불안감을 해소하는 것이 최우선
   - 공정거래위원회 정보공개서 기반 데이터, 실제 점주 인터뷰, 구체적 수익 사례 필수

3. **모바일 퍼스트 설계**
   - 40대 퇴직자도 유튜브·네이버를 모바일로 소비. 랜딩페이지는 모바일 최적화 필수

4. **CTA 차별화**
   - 탈직장인: "수익 시뮬레이션 받기" (계산기형 인터랙티브 도구)
   - 다점포 운영자: "가맹점 실적 자료 요청" (비즈니스 자료 다운로드)
   - 소자본 첫 창업: "무료 창업 상담 신청" (낮은 진입 장벽)

### 6.2 메시지 톤 시사점

- **과장을 피하고 데이터로 설득**: "월 1,000만 원 수익!" 같은 과장보다 "가맹점 평균 매출 2,669만 원, 순이익률 XX%" 같은 팩트 제시
- **실패 리스크를 인정하고 대응책을 제시**: "창업이 쉽다"가 아니라 "어렵지만, 이렇게 지원합니다"
- **동류 집단의 증언 활용**: 비슷한 배경의 실제 가맹점주 스토리가 가장 강력한 설득 도구
