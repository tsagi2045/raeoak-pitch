# 수도권 소형 쌀국수 경쟁 현황 조사

> 조사일: 2026-04-13
> 소스: 웹 서치 (마이프차, 다이닝코드, 나무위키, 한국경제, 겟뉴스, 각 브랜드 공식 사이트)
> 목적: 라이옥 라이트의 "정통+소형+카운터" 포지셔닝이 실제로 비어있는지 검증

---

## 1. 조사 배경

라이옥 라이트 리브랜딩 진단에서 "정통 베트남 육수 + 소형 밀착 경험" 슬롯이 비어있다고 가설을 세움. 이를 검증하기 위해 수도권 쌀국수 프랜차이즈 시장을 조사.

## 2. 경쟁 브랜드 상세

### 2.1 미분당

| 항목 | 내용 |
|------|------|
| 매장 수 | ~171개 (업계 1위) |
| 창업 연도 | 2014년 서울 서대문구 |
| 법인 전환 | 2018년 프랜차이즈 사업 시작 |
| 포지셔닝 | **한국식 쌀국수**, 감성 공간 |
| 좌석 형태 | **오픈형 주방을 마주한 바(Bar) 형태 좌석** — 혼밥 고객 편안함. 일반 테이블석도 병행 |
| 인테리어 | 일본식 카운터석, 천장 갓등, 목재 소스 받침대. 인테리어 독자적 식별력 법원 인정 (미분당 vs 월미당 소송 승소, 2023) |
| 가격대 | 중간~중상 |
| 핵심 차별점 | 감성 인테리어 + 한국인 입맛에 맞춘 쌀국수 |
| 라이옥 라이트와의 차이 | **정통 베트남이 아닌 한국식**. 카운터석은 있으나 "정통성"이 아닌 "감성"이 포인트 |

**시사점**: 미분당이 카운터/바좌석 + 쌀국수를 이미 하고 있으므로, "카운터 쌀국수"라는 표현만으로는 차별화 부족. **"정통 베트남 육수"가 핵심 차별점**이 되어야 함.

### 2.2 리틀하노이

| 항목 | 내용 |
|------|------|
| 매장 수 | ~60개 (입소문 기반) |
| 직영 매장 | 4개 — 종로점(50평), 부평점(35평), 신정점(25평), 광화문점(10평) |
| 포지셔닝 | "KOREAN STYLE VIETNAM NOODLE" — 한국식 조리법 응용 |
| 가격 | **3,900원** (초저가) |
| 메뉴 | 7가지 메뉴, 높은 품질+낮은 가격 |
| 매장 규모 | 10평~50평 다양 |
| 라이옥 라이트와의 차이 | **한국식, 초저가**. 정통 베트남이 아님. 가격대가 완전히 다름 (3,900원 vs 라이옥 11,000원) |

**시사점**: 소형 쌀국수 매장은 이미 존재하지만, "초저가 한국식"에 집중. 정통+적정 가격대의 소형 모델은 부재.

### 2.3 꾸아 (QUA)

| 항목 | 내용 |
|------|------|
| 매장 수 | ~29개 |
| 운영사 | 찹찹푸드 |
| 포지셔닝 | **하노이식 정통 쌀국수**, 베트남 캐주얼 다이닝 |
| 대표 메뉴 | 3단 트레이 반쎄오, 하노이식 정통 쌀국수, 직화쌀국수 |
| 매장 형태 | **다이닝 스타일** — 테이블 중심, 성수·건대 등 |
| 특이사항 | 투자자문사에 **매각 관련 페이지 존재** (모멘스투자자문) — 매각 절차 진행 가능성 |
| 라이옥 라이트와의 차이 | 정통성은 유사하나 **대형 다이닝 모델**. 소형/카운터가 아님. 매각 가능성 있으면 브랜드 약화 가능 |

**시사점**: "정통 하노이식" 포지션의 직접 경쟁자이나, 매장 형태와 규모가 다름. 매각 진행 시 해당 포지션에 공백 발생 가능.

### 2.4 호앙비엣 (매장 형태 확인 완료: 2026-04-13)

| 항목 | 내용 |
|------|------|
| 매장 수 | ~20개 (수도권 중심) |
| 확장 속도 | **6개월간 11개 신규 오픈** (급성장) |
| 포지셔닝 | 정통 베트남, **현지 공장 직제조 면·소스**, 자체 물류 |
| 매장 형태 | **2층 구성, 넓은 매장** (동국대점: 서울 중구 서애로1길 14, 1·2층) |
| 1층 좌석 | 바 테이블(기다란 좌석) + 일반 테이블 혼합. 혼밥 가능. 스테인리스 테이블 |
| 2층 좌석 | 테이블 2개 연결, 4인+ 그룹 대응 |
| 인테리어 | 베트남 현지 골목 식당 감성. 노란 간판, 베트남 전통 갓 장식, 패턴 타일 |
| 주문 방식 | 종이 체크식 (종이에 메뉴 체크 → 직원에게 전달) |
| 가격대 | 쌀국수 ₩10,500~₩12,000, 반쎄오 ₩3,500/개, 플래터 ₩18,000~₩19,000 |
| 리뷰 키워드 | 현지감성, 깔끔함, 가성비, 혼밥 최적화, 이국적, 깊은 국물 맛 |
| 라이옥 라이트와의 차이 | 정통성은 유사하나 **2층 규모의 대형 모델**. 바 좌석은 일부 있으나 매장 전체가 소형 카운터 중심은 아님 |

**시사점**: 매장 형태 확인 결과 **"정통+대형" 모델**로 판명. 바 좌석이 일부 있지만 2층 규모의 일반 매장이므로, 라이옥 라이트의 "정통+소형+카운터" 교집합에 해당하지 않음. **포지셔닝 전제 유효.**
다만 가격대(₩10,500~₩12,000)가 라이옥(₩11,000)과 거의 동일하고 정통성을 강하게 소구하므로, "정통 베트남 육수" 단독으로는 호앙비엣과 차별화 어려움. **"소형 밀착 경험"이 차별의 열쇠.**

## 3. 멘탈 슬롯 매트릭스

### 3.1 2축 매핑: 정통성 × 매장 규모

```
정통 베트남 ↑
             │  꾸아(29)        포메인(100)
             │     호앙비엣(20)
             │
             │  ★ 라이옥 라이트  ← 이 교집합이 비어있음
             │
─────────────┼────────────────────────────→
소형/카운터    │                    대형/테이블
             │
             │  리틀하노이(60)    미분당(171)
             │  배달의쌀국수(67)   에머이(37)
             │
한국식/퓨전   ↓
```

### 3.2 결론

**"비어있는 슬롯"이라는 주장은 조건부로 유효하다:**

- ❌ "카운터 쌀국수가 비어있다" → 미분당이 이미 점유
- ❌ "정통 베트남이 비어있다" → 꾸아, 호앙비엣이 점유
- ❌ "소형 쌀국수가 비어있다" → 리틀하노이, 배달의쌀국수 다수
- ✅ **"정통 베트남 + 소형 카운터"의 교집합은 비어있다**

## 4. 라이옥 라이트 포지셔닝 시사점

1. **"카운터 쌀국수"만으로는 차별화 불가** — 미분당과 겹침. "정통 베트남 육수"가 반드시 앞에 와야 함.
2. **가격대로 리틀하노이와 분리** — 리틀하노이 3,900원 vs 라이옥 11,000원. 같은 소형이지만 완전히 다른 세그먼트.
3. **호앙비엣이 가장 위협적** — 정통성+수도권 급확장. 매장 형태 실사 확인이 시급.
4. **꾸아 매각 가능성** — 사실이라면 "정통 하노이식" 포지션에 공백 → 라이옥에 기회.
5. **포지셔닝 공식 수정**: "정통+소형+카운터" 교집합이 비어있음을 근거로, 개별 요소 경쟁자와의 차이를 명확히 하는 방향.
