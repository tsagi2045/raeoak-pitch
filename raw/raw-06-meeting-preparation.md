## 2. 질문 목록

### A. 브랜드 및 사업 현황 (6문항)

| # | 질문 | 의도 |
|---|------|------|
| A1 | 브랜드의 시작 스토리와 핵심 철학은 무엇인가요? | 랜딩페이지 헤드라인/스토리텔링 소재 확보 |
| A2 | 다른 쌀국수 프랜차이즈 대비 가장 큰 차별화 포인트는? | USP(Unique Selling Proposition) 도출 |
| A3 | 현재 직영점/가맹점 수는 각각 몇 개이며, 지역 분포는? | 브랜드 규모와 성장 단계 파악 |
| A4 | 정보공개서 기준 가맹점 평균 연매출과 영업이익률은? | 랜딩페이지 수익 데이터 활용 가능 범위 확인 |
| A5 | 최근 3년 폐점률과 가맹점 순증 추이는? | 안정성 근거 확보, 미스사이공 사례(274→90) 대비 포지셔닝 |
| A6 | 올해/내년 가맹점 모집 목표 수는? | SOM 재산출 및 랜딩페이지 KPI 설정 기준 |

### B. 가맹 조건 및 지원 체계 (6문항)

| # | 질문 | 의도 |
|---|------|------|
| B1 | 초기 투자비 구성은? (가맹비, 교육비, 보증금, 인테리어비 등) | 투자비 투명 공개 여부 결정 — 경쟁사 대부분 비공개이므로 공개 시 차별화 가능 |
| B2 | 로열티 구조는? (정액/정률, 월 금액) | 가맹점 수익 시뮬레이션에 필요 |
| B3 | 소형 매장/배달 특화 모델이 있는가? 최소 평수는? | 소자본 창업자(페르소나 C) 대응 가능 여부 확인 |
| B4 | 교육 프로그램 상세는? (기간, 내용, 장소) | 랜딩페이지 "교육/지원" 섹션 콘텐츠 |
| B5 | 슈퍼바이저 지원 체계는? (방문 주기, 지원 범위) | 본사 신뢰도 근거 — 1순위 의사결정 기준 |
| B6 | 상권 분석/입지 선정 지원을 본사에서 하는가? | 포메인 AI 입지 분석 대비 차별화 또는 벤치마킹 포인트 |

### C. 현재 가맹 모집 현황 (6문항)

| # | 질문 | 의도 |
|---|------|------|
| C1 | 현재 가맹점 모집은 어떤 채널로 하고 있나요? (온라인/오프라인) | 기존 퍼널 파악 및 랜딩페이지의 위치 설정 |
| C2 | 기존 가맹 모집 페이지나 랜딩페이지가 있나요? URL? | 기존 자산 활용 또는 리뉴얼 범위 결정 |
| C3 | 월 평균 가맹 문의 건수는? (전화, 온라인, 박람회 등) | 현재 리드 볼륨 파악 → 랜딩페이지 목표 설정 |
| C4 | 문의에서 계약까지의 전환율은 대략 몇 %인가요? | 전환 퍼널 벤치마크 — 고객 여정 분석(05)의 전환율 데이터와 비교 |
| C5 | 가맹점주 유입 경로 데이터가 있나요? (어떤 채널에서 문의?) | 광고 채널 전략 수립 기초 데이터 |
| C6 | 기존 가맹점주 중 가장 많은 유형은? (연령, 이전 직업, 창업 동기) | 타겟 페르소나(03) 검증 — Primary 타겟이 탈직장인이 맞는지 확인 |

### D. 마케팅 자산 및 소재 (5문항)

| # | 질문 | 의도 |
|---|------|------|
| D1 | 브랜드 가이드라인이 있나요? (로고, 컬러, 폰트, 톤앤매너) | 랜딩페이지 디자인 기준 |
| D2 | 매장/음식 고해상도 사진, 영상 소재가 있나요? | 비주얼 콘텐츠 확보 |
| D3 | 가맹점주 인터뷰나 후기 영상이 있나요? | 사회적 증거(Social Proof) — 전환율에 가장 큰 영향 |
| D4 | 수상 이력, 언론 보도 자료가 있나요? | 신뢰 배지 (Trust Badge) 활용 |
| D5 | 재무 데이터(매출, 이익률 등) 랜딩페이지 공개 범위는 어디까지? | 투명성 전략의 핵심 — 경쟁사 대부분 비공개이므로 공개 수준이 곧 차별화 |

### E. 랜딩페이지 요구사항 (5문항)

| # | 질문 | 의도 |
|---|------|------|
| E1 | 랜딩페이지에서 가장 강조하고 싶은 메시지 1~3가지는? | 클라이언트의 USP 인식과 리서치 결과 비교 |
| E2 | 문의 폼에 어떤 정보를 받고 싶나요? (이름, 연락처, 희망 지역, 가용 자산 등) | 폼 필드 설계 — 필드가 많을수록 전환율 하락, 적을수록 리드 품질 하락 |
| E3 | 문의 접수 후 프로세스는? (자동 응답 → 담당자 연결 → 상담 일정?) | CTA 후 퍼널 설계 |
| E4 | 기존 웹사이트, CRM, 마케팅 도구와 연동이 필요한가요? | 기술적 요구사항 파악 |
| E5 | 런칭 희망 일정과 예산 범위는? | 프로젝트 스코프 및 일정 설정 |

## 3. 사전 리서치 핵심 발견사항 (미팅 중 공유용)

미팅 중 50~60분 지점에서 사전 리서치 결과를 간략히 공유하여 클라이언트의 피드백을 받습니다.

### 발견 1: 쌀국수 프랜차이즈는 아직 블루오션

> "치킨 3만개, 커피 4만개 시대에 쌀국수는 전체 600~700개 수준. 에스닉 푸드 매장이 9년간 5배 성장했고, 건강식/1인 가구 트렌드와 정확히 맞물립니다."

- **클라이언트에게 질문**: 이 '블루오션' 메시지를 랜딩페이지 핵심 메시지로 활용해도 괜찮은지?

### 발견 2: 경쟁사 대부분 독립 랜딩페이지가 없음

> "포메인(phomeinfc.com)만 유일하게 가맹 전용 독립 사이트를 운영합니다. 나머지는 홈페이지 내 서브 섹션. 잘 만든 랜딩페이지 하나만으로도 차별화가 가능합니다."

- **클라이언트에게 질문**: 독립 도메인(예: 브랜드명franchise.com) vs 기존 사이트 내 랜딩 중 선호하는 방식은?

### 발견 3: '투명성'이 최대 차별화 무기

> "대부분 경쟁사가 가맹비/창업비용을 공개하지 않습니다. 투자비 범위와 정보공개서 기반 매출 데이터를 공개하면 '정보를 숨기지 않는 프랜차이즈'로 포지셔닝할 수 있습니다."

- **클라이언트에게 질문**: 투자비와 매출 데이터의 공개 가능 범위는?

### 발견 4: Primary 타겟은 '탈직장인 창업자'

> "30~50대가 전체 창업의 71.4%. 퇴직금 1~2억을 가진 탈직장인이 가장 큰 세그먼트이며, 쌀국수 프랜차이즈 투자 규모(1~1.5억)와 정확히 부합합니다."

- **클라이언트에게 질문**: 실제 가맹점주 중 이 유형이 가장 많은지? 다른 주요 유형이 있는지?

### 발견 5: 전환의 핵심은 '배우자/가족 설득'

> "가맹 창업은 개인 결정이 아닌 가족 공동 의사결정입니다. 탈직장인 창업자의 경우 배우자 설득이 최대 전환 장벽이 됩니다."

- **클라이언트에게 질문**: 가맹 상담 시 배우자 동반이 많은지? 가족 설득용 자료가 별도로 있는지?

## 5. 미팅 중 유의사항

### Do (해야 할 것)

| 항목 | 이유 |
|------|------|
| 리서치 결과를 **질문 형태**로 제시 | "저희가 조사한 바로는 ~인데, 실제로는 어떤가요?" — 전문성 + 경청 |
| **숫자**로 근거 제시 | "치킨 3만개 대비 쌀국수 600개" — 구체적 데이터가 설득력 |
| 경쟁사 랜딩페이지 **캡처**를 보여주며 설명 | 시각적 레퍼런스가 방향 합의에 효과적 |
| **미결 사항**을 명시적으로 기록 | "이건 자료 확인 후 결정"으로 넘기고, 미결 리스트 유지 |
| 클라이언트의 **직감/경험** 존중 | 데이터와 현장 경험이 충돌하면 현장이 우선일 수 있음 |

### Don't (하지 말 것)

| 항목 | 이유 |
|------|------|
| 경쟁사를 **비하**하지 않기 | "미스사이공이 망했다"가 아니라 "급속 확장의 리스크 사례" |
| 구체적 **매출 약속**을 하지 않기 | 정보공개서 외 데이터로 수익을 보장하면 법적 리스크 |
| **기술 용어** 남용하지 않기 | CTA, 퍼널, A/B 테스트 → "문의 버튼", "고객 동선", "비교 실험" |
| 일정을 **너무 빡빡하게** 잡지 않기 | 여유를 두어 품질 확보 |

## 7. 클라이언트 사전 공유용 요약 (1페이지)

> 미팅 2일 전, 클라이언트에게 아래 내용을 요약하여 공유합니다.  
> 목적: "이 팀이 준비를 잘 해왔구나"라는 인상 + 미팅 방향에 대한 사전 합의

---

### 쌀국수 프랜차이즈 가맹점 모집 랜딩페이지 — 사전 리서치 요약

**1. 시장 기회**
- 에스닉 푸드 매장 9년간 5배 성장, 외식업 경기전망지수 1위 카테고리
- 쌀국수 프랜차이즈 전체 600~700개 수준 — 치킨(3만+), 커피(4만+) 대비 초기 성장 단계
- 건강식(글루텐프리, 저칼로리), 1인 가구(815만), 배달 시장(40조) 트렌드와 높은 정합성

**2. 경쟁 환경**
- 주요 브랜드: 미분당(171개), 포메인(100개), 미스사이공(90개) 등 10여 개
- 가맹 전용 독립 랜딩페이지 운영 브랜드는 포메인 1곳뿐
- 대부분 가맹비/창업비용을 웹상에 비공개 → **투명성이 곧 차별화**

**3. 타겟 가맹점주**
- 핵심 타겟: 30~50대 탈직장인 (전체 창업의 71.4%, 투자금 1~2억 보유)
- 의사결정 기준: ① 수익 안정성 ② 본사 신뢰도 ③ 운영 난이도
- 핵심 전환 장벽: 배우자/가족 설득, 투자금 손실 두려움

**4. 랜딩페이지 방향 (가설)**
- 모바일 퍼스트, 가맹 전용 독립 페이지
- 투자비 범위 + 정보공개서 기반 매출 데이터 투명 공개
- 실제 가맹점주 인터뷰 + 수익 사례 중심 설득
- 간편 상담 신청 폼 → 자동 안내 → 1:1 상담 연결 퍼널

**미팅에서 논의하고 싶은 것**
- 브랜드 차별화 포인트와 핵심 메시지 방향
- 투자비/매출 데이터 공개 가능 범위
- 현재 가맹 모집 현황과 문의 데이터
- 런칭 일정과 우선순위

## 8. 가맹사업법 주요 체크포인트

랜딩페이지 콘텐츠 작성 시 법적 리스크를 사전에 방지하기 위해 확인이 필요한 항목입니다.

| 항목 | 내용 | 미팅 시 확인 |
|------|------|-------------|
| **정보공개서 등록** | 가맹사업법상 정보공개서를 공정위에 등록해야 가맹점 모집 가능 | 등록 완료 여부 및 최신 갱신일 |
| **가맹금 예치** | 가맹금 예치제도 — 가맹비 등 선수금을 예치기관에 예치 | 예치 기관 및 절차 안내 여부 |
| **허위/과장 광고 금지** | 수익 보장, 확정 매출 등의 표현은 가맹사업법 위반 가능 | 랜딩페이지 문구 사전 법률 검토 필요 여부 |
| **정보공개서 제공 의무** | 가맹 희망자에게 정보공개서를 14일 전 제공해야 함 | 랜딩페이지에서 정보공개서 다운로드 제공 가능 여부 |
| **2025년 가맹사업법 개정** | 직영점 3개 이상 필수, 정보 투명성 강화 | 개정법 충족 여부 확인 |
